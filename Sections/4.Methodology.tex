\chapter{實驗結果}
\label{ch:results}

本章根據正式實驗結果，回答本文的核心研究問題：在台股橫斷面報酬預測中，模型結構、圖拓樸與訓練視窗長度如何影響 rank signal 與投資組合表現。完整實驗共 27 組，包含 short、medium、long 三種訓練視窗，以及 Linear、XGBoost、LSTM、DNN、GAT-industry、GAT-universe、GAT-TwoGraph、GAT-IndNeutral 與 GAT-full 九類模型。Baseline 模型不使用圖結構；GAT 與 GAT 中性化類模型則使用相同 artifacts，以避免不同資料切分或圖重估方式干擾模型比較。

本文以 Daily IC 與 ICIR 作為主要 signal quality 指標，因本研究任務為橫斷面排序而非單純數值迴歸。投資組合績效則用以檢驗排序訊號是否能轉化為 top 10\% 等權做多策略的可用報酬。為避免單一指標的選擇偏誤，以下分析同時考量訓練視窗、圖拓樸、train-test gap 與投組風險，並於各小節分別討論。

\section{市場基準與勝出模型摘要}
\label{sec:static_overview}

表 \ref{tab:results_benchmark_0050} 先列出三個測試視窗中 0050 基準的市場背景。Short 測試期為 2024-12-03 至 2025-05-29，0050 年化報酬為 -5.12\%、Sharpe 為 -0.150，且最大跌幅達 -27.37\%；Medium 與 Long 測試期則分別呈現 25.52\% 與 24.12\% 的 0050 年化報酬。三個測試視窗對應不同市場環境，因此不宜直接將 short window 的結果外推為長期結論。

\input{Sections/tables/results_benchmark_0050.tex}

表 \ref{tab:results_pred_metrics} 彙整各視窗的勝出模型。Short window 中，DNN 取得最高 Daily IC（0.0396）與 ICIR（0.408），但最佳 Sharpe 仍僅為 GAT-industry 的 0.057，且所有模型的總報酬皆為負。Medium window 開始出現較可用的投組表現：GAT-full 取得最高 Daily IC（0.0596）與 ICIR（0.628），DNN 則取得最高 Sharpe（0.980）與最高總報酬（39.13\%）。Long window 是結果最穩定的設定，GAT-IndNeutral 取得最高 Daily IC（0.0590）與 ICIR（0.786），GAT-full 取得最高 Sharpe（1.849），XGBoost 則取得最高總報酬（143.08\%）。整體而言，GAT 類模型未呈現全面勝出；模型表現高度取決於訓練視窗、圖拓樸與評估層級。

\section{訓練視窗長度對 out-of-sample 可用性之影響}
\label{sec:window_length_results}

表 \ref{tab:results_portfolio_metrics} 以模型家族為單位比較三個訓練視窗。結果顯示，訓練資料長度對所有模型皆有明顯影響，但對圖模型與 GAT 中性化類模型尤其關鍵。Short window 的訓練樣本相對有限，雖然多數模型的 Daily IC 已轉正，但投組年化報酬多數接近 0 或為負；例如 DNN 的 Daily IC 為 0.0396，Sharpe 卻為 -0.098，GAT neutralization family 的平均 Daily IC 為 0.0278，平均 Sharpe 也僅 -0.084。換言之，短樣本下的弱正向排序訊號尚不足以穩定轉化為可投資報酬。

\input{Sections/tables/results_portfolio_metrics.tex}

Medium window 擴大訓練樣本後，模型表現明顯改善。GAT neutralization family 的平均 Daily IC 提升至 0.0548，平均 Sharpe 提升至 0.873；DNN 亦達到 0.980 Sharpe。Long window 的改善更穩定：GAT neutralization family 平均 Daily IC 為 0.0587、平均 ICIR 為 0.760、平均 Sharpe 為 1.826；GAT family 平均 Daily IC 為 0.0502、平均 Sharpe 為 1.704；DNN 則為 Daily IC 0.0535、Sharpe 1.564。這表示，在本資料中，long window 比 short/medium 更能覆蓋不同市場 regime，使模型學到的 rank signal 在 out-of-sample 投組層級較穩定。

此結果亦揭示一項方法論考量：若僅以 short window 進行模型比較，易低估 GAT 與 GAT 中性化類模型的潛在效益；若僅觀察 long window 的投組報酬，則可能高估 XGBoost 等高容量 tabular 模型的泛化穩健性。較完整的評估方式應同時報告三個視窗，並以 IC / ICIR 與投組績效共同衡量。

\section{圖拓樸設計對模型排序效益之影響}
\label{sec:topology_results}

表 \ref{tab:results_static_long_topology} 進一步比較 long window 下的模型結構與圖拓樸。最直接的觀察是，GAT-universe 並未優於 DNN：GAT-universe 的 Daily IC 為 0.0452、Sharpe 為 1.506，低於 DNN 的 Daily IC 0.0535 與 Sharpe 1.564。這表示全市場完全連結並不必然提供有效增量；若每檔股票都與所有股票交換訊息，模型可能過度吸收市場共同成分，反而降低個股截面差異。

\input{Sections/tables/results_static_long_topology.tex}

相對地，GAT-industry 在 long window 的 Daily IC 為 0.0543、Sharpe 為 1.779，皆高於 GAT-universe；在 medium window 中亦有相同方向，GAT-industry 的 Daily IC 為 0.0445、Sharpe 為 0.900，高於 GAT-universe 的 0.0377 與 0.755。這支持「經濟意義圖拓樸有效」的論點：產業圖提供的是相對穩定的局部關係，使模型能比較同業內的相對強弱；universe graph 則近似全連接，容易把市場共同波動與風格共動混入個股訊號。

GAT-TwoGraph 的結果呈現另一種 trade-off。Long window 下，其 Daily IC 為 0.0512，低於 GAT-industry 的 0.0543，但 Sharpe 提升至 1.827，且最大跌幅為 -19.23\%，是表中最低。這代表雙圖訊息在較長樣本下可改善投組風險報酬，但不一定同步提升平均 Daily IC。換言之，圖拓樸的價值不應只用「是否使用 GAT」判斷，而應拆成 industry graph、universe graph 與 dual graph 三種不同資訊結構。

\section{GAT 中性化與 factor attention 之預測品質分析}
\label{sec:dmfm_results}

GAT 中性化類模型在 medium 與 long window 的 IC / ICIR 表現最為突出。Medium window 中，GAT-full 取得全模型最高 Daily IC（0.0596）與 ICIR（0.628）；Long window 中，GAT-IndNeutral 取得最高 Daily IC（0.0590）與 ICIR（0.786），GAT-full 亦有 Daily IC 0.0583 與 Sharpe 1.849。這比只看年化報酬更符合橫斷面 alpha 模型的評估邏輯，因為 IC / ICIR 直接衡量模型是否穩定把相對強勢股票排在前段。

\input{Sections/tables/results_dmfm_tradeoff.tex}

表 \ref{tab:results_dmfm_tradeoff} 顯示，GAT-full 與 GAT-IndNeutral 的差異可解讀為 signal stability 與 portfolio return 的權衡。GAT-IndNeutral 的 ICIR（0.786）與產業中性 ICIR（0.753）較高，最大跌幅也較小（-20.11\%）；GAT-full 則有較高 Test IC（0.0602）、Sharpe（1.849）、年化報酬（42.02\%）與勝率（68.75\%）。這表示 GAT-IndNeutral 較像純 alpha rank signal，GAT-full 則保留較多產業或市場 exposure，在有利市場環境下能轉化為較高投組報酬。

注意力特徵亦提供補充證據。表 \ref{tab:results_dmfm_attention} 顯示，Short window 較常出現 \texttt{down\_vol\_20}、\texttt{roll\_max\_10}、\texttt{roll\_min\_60}、\texttt{roll\_max\_5} 與 \texttt{pct\_to\_low\_20}，集中於近期價格位置、下行波動與短期技術訊號；Medium window 則以 \texttt{atr\_14}、\texttt{roll\_min\_5}、\texttt{roll\_max\_20}、\texttt{macd} 與 \texttt{std\_ret\_20} 為主；Long window 中，\texttt{ps} 與 \texttt{macd\_signal} 開始進入高頻特徵，並搭配 rolling min/max 與 \texttt{atr\_14}。這與金融直覺一致：短樣本較依賴近期價格位置與波動訊號，長樣本才較能穩定吸收估值與較長週期風險特徵。不過，attention weight 仍只能作為描述性解釋，不能直接視為因果效果。

\section{Baseline 模型表現診斷}
\label{sec:baseline_diagnostics}

Baseline 模型的結果提供本文主張的重要對照基準。Linear 的整體表現最弱，三個視窗平均 Daily IC 僅 0.0168，且 short window Sharpe 為 -0.423，顯示 56 維特徵中存在非線性與交互作用，單純線性映射不足以穩定轉化為投組報酬。LSTM 在 long window Sharpe 達 1.336，較 Linear 有所改善，但仍低於 DNN、GAT family 與 GAT neutralization family，顯示單股時間序列記憶並不足以取代橫斷面非線性特徵與關係結構。

DNN 在本文所有非圖基準中最具參考價值。儘管不使用任何圖結構，其在 short window 取得最高 Daily IC（0.0396）與 ICIR（0.408），在 medium window 取得最高 Sharpe（0.980）與總報酬（39.13\%），在 long window 亦達 Daily IC 0.0535 與 Sharpe 1.564。此結果說明特徵工程本身已提供相當程度的排序訊號；圖模型的增益應被定位為在較長樣本下的穩定性、拓樸可解釋性與增量貢獻，而非對非圖模型的全面性主導優勢。

\input{Sections/tables/results_xgboost_gap.tex}

XGBoost 呈現高總報酬但伴隨明顯 train-test 差距。表 \ref{tab:results_xgboost_gap} 顯示，其 long window 總報酬達 143.08\%，為所有模型最高，但 Train IC 為 0.7824 而 Test IC 僅 0.0205，train/test 比達 38.1 倍，顯示在投組層級表現背後存在顯著的過擬合風險。本文將 XGBoost 定位為強 tabular baseline，但其 signal robustness 與泛化能力需一併揭露，不宜以單一總報酬指標作為最終評判依據。

\section{評估指標與限制}
\label{sec:evaluation_limits}

本實驗也顯示，MSE / RMSE 不適合作為跨架構的唯一比較指標。Long window 中，Linear、LSTM 與 GAT-industry 的 RMSE 約 0.0484--0.0485，但 DNN、GAT-IndNeutral 與 GAT-full 的 RMSE 分別為 0.1336、0.1160 與 0.1817；然而後三者在 Daily IC、ICIR 或 Sharpe 上並不差。原因在於本文模型的主要用途是截面排序，且部分 factor-attention / neutralization 模型的輸出尺度與原始報酬空間不完全一致。只要排序能穩定把相對強勢股票放在前段，較高 RMSE 不必然代表較差投組表現。

本實驗存在若干重要限制，應於解讀結果時一併考慮。其一，現階段回測尚未納入交易成本、滑價、容量限制與換手率控制，現有績效數字應視為無摩擦假設下的上限估計。其二，尚未進行分年度、分市場 regime、paired $t$-test 或 bootstrap 穩健性檢定，模型優劣的跨樣本一致性有待進一步驗證。其三，圖建構所使用之產業分類資訊屬穩定外生資訊，但若邊或權重引入全樣本資訊，則需重新以訓練期資料建立圖並進行 look-ahead bias 檢查。基於上述考量，本文較穩健的結論為：訓練視窗長度與圖拓樸設計是台股橫斷面 rank signal 的關鍵影響因素；在 medium 與 long window 下，GAT 中性化系列與 industry-aware GAT 類模型展現較穩定的排序效益；naive universe graph 與線性 baseline 表現相對較弱；DNN 與 XGBoost 的結果則說明，圖模型的增益主要體現在穩定性與可解釋性的提升。
