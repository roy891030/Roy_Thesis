\chapter{實驗結果}
\label{ch:results}

本章根據 \texttt{runs\_unified/analysis\_report} 的統一彙整結果，回答本文的核心研究問題：在相同 TEJ 日頻資料、相同時間切分與相同回測設定下，圖注意力、多因子表示、分層中性化與 dynamic graph 是否能提升台股截面選股訊號。完整實驗共 45 組，包含 static / dynamic 兩種圖模式、Short / Medium / Long 三種樣本視窗、三個非圖 baseline，以及六個 factor ablation models。需特別說明，baseline 不使用圖結構，因此預設僅列於 static graph 口徑；dynamic graph 的比較主要適用於六個 factor variants。

\section{預測品質指標結果}
\label{sec:prediction_quality_results}

表 \ref{tab:results_prediction_quality_full} 依第三章「預測品質指標」的定義，完整列出各模型在測試集上的 MSE、RMSE、MAE、IC、Daily IC、ICIR、DirAcc 與產業中性 IC。這張表主要用來觀察兩件事：第一，誤差指標是否真的能反映截面排序能力；第二，模型在 Daily IC 與 ICIR 上是否具備跨日穩定性。由於本文的投資任務是依預測分數做截面排序，IC / Daily IC / ICIR 的解讀權重高於單純 error metrics；但 MSE、RMSE 與 MAE 仍保留，用以檢查模型是否以過大的數值偏差換取排序改善。

\input{Sections/tables/results_prediction_quality_full.tex}

由表 \ref{tab:results_prediction_quality_full} 可見，非圖 baseline 的表現已呈現明確分層。Linear 在三個 window 的 Test IC 分別為 0.0255、0.0200 與 0.0396，代表單純線性映射雖可提供可解釋下限，但難以充分吸收 56 維技術特徵中的非線性關係。XGBoost 則是 baseline 中最穩健的強基準：Short/static 的 IC 為 0.0704、ICIR 為 1.044，Long/static 的 IC 亦達 0.0856，顯示 tree-based model 對 tabular technical features 的非線性切分與交互作用具有實質幫助。LSTM 的 IC 在三個 window 分別為 0.0491、0.0707 與 0.0704，整體高於 Linear 但多數情況低於 XGBoost；這表示單股十日 lookback 序列確實提供時間資訊，但在本研究的截面選股任務中，單股時間記憶尚不足以取代非線性 tabular tree 或跨股票關係建模。

Factor variants 則進一步說明圖結構與因子表示的增量效果。Short/static 中 GAT-Universe 取得最高 IC（0.0843）與 Daily IC（0.0843），相較 XGBoost 的 0.0704 有明顯提升，代表在短樣本中，全市場關係聚合能補充 tabular features 的截面訊號。Medium/static 的最高 IC 由 GAT-Industry 取得（0.0933），且產業中性 IC 為 0.0976，表示其排序能力並非只來自產業平均差異，而是在產業內去均值後仍保留個股相對強弱。Long/dynamic 中 GAT-Universe 的 IC 達 0.0955，是全表最高，說明在較長樣本下，rolling correlation 所建立的全市場關係對截面排序最有幫助。

同時，表中也顯示 error metrics 與排序指標不完全一致。例如 Long/static 的 DMFM-Full RMSE 為 0.1349，明顯高於 XGBoost 的 0.0556，但其投組層面反而取得最高總報酬與 Sharpe（見表 \ref{tab:results_portfolio_0050_full}）。這表示回歸誤差較低不必然代表 top portfolio 較佳，因為投資組合只使用排序前 10\% 的股票；只要模型能穩定把相對強勢股票排到前段，即使對所有股票的絕對報酬數值擬合較差，仍可能在策略層面產生較高績效。DirAcc 亦需謹慎解讀：部分模型 IC 高但 DirAcc 偏低，代表模型更像排序器而非方向分類器，這與本文採用截面去均值標籤的設計一致。

\section{投資組合績效與 0050 基準比較}
\label{sec:portfolio_0050_results}

表 \ref{tab:results_benchmark_0050} 先列出三個測試區間中 0050 的基準績效。這張表的目的不是比較模型，而是提供後續策略績效的市場背景。Short 測試區間僅有 13 次再平衡，0050 總報酬為 21.0\%、Sharpe 為 4.201，屬於非常強的上行期間；Medium 測試區間 0050 總報酬為 -7.5\%、Sharpe 為 -0.329，市場基準偏弱；Long 測試區間 0050 總報酬為 18.2\%、Sharpe 為 0.625，但最大回撤達 -24.0\%，反映較長期間內市場風險明顯提高。

\input{Sections/tables/results_benchmark_0050.tex}

表 \ref{tab:results_portfolio_0050_full} 則依第三章「投資組合績效指標」與「基準比較」的框架，完整列出策略年化報酬、總報酬、Sharpe、Hit ratio、最大回撤、0050 同期總報酬、0050 Sharpe、超額總報酬與超額 Sharpe。這張表的關鍵是把模型訊號轉換為可投資排序後，再觀察其是否能在相同持有期與再平衡頻率下超越被動市場基準。

\input{Sections/tables/results_portfolio_0050_full.tex}

Short window 的解讀必須先注意基準本身很強：0050 在 13 次再平衡中已有 21.0\% 總報酬，因此 Linear（17.3\%）與 LSTM（20.5\%）雖然 Sharpe 高，總報酬仍低於 0050。XGBoost 則以 31.1\% 總報酬與 7.608 Sharpe 顯著超越 0050，說明強 tabular model 在短樣本中已具備可投資性。圖模型方面，GAT-Universe/static 取得全 Short window 最佳總報酬（32.1\%）與 Sharpe（7.832），超額總報酬為 11.2 個百分點；dynamic 設計中 DMFM-Full 的總報酬為 27.7\%，GAT-TwoGraph 的 Sharpe 為 7.318，代表 dynamic graph 仍可產生穩健策略，但在短期強勢市場中未全面超越 static universe graph。

Medium window 則呈現完全不同的市場背景。0050 總報酬為 -7.5\%，而所有模型策略皆為正報酬，表示 top 10\% 截面排序在弱勢市場中具備防禦與相對選股效果。Linear 的總報酬僅 0.7\%，雖然已高於 0050，但經濟意義有限；LSTM 與 XGBoost 分別提升至 22.7\% 與 14.4\%。Factor variants 的優勢在此區間更明顯：GAT-Industry/static 與 GAT-TwoGraph/static 的總報酬皆約 36.1\%，dynamic GAT-Universe 則進一步達 38.3\%、Sharpe 1.866、超額總報酬 45.8 個百分點。這個結果與表 \ref{tab:results_prediction_quality_full} 中 Medium window 的高 IC 相互呼應，說明模型的截面排序能力在市場下行或震盪期間更容易轉化為基準相對優勢。

Long window 中，0050 雖有 18.2\% 總報酬，但 Sharpe 僅 0.625 且最大回撤達 -24.0\%。所有策略皆高於 0050 總報酬，其中 XGBoost 已達 48.4\%，顯示強 baseline 在長樣本仍具競爭力。最佳策略為 DMFM-Full/static，總報酬 54.3\%、Sharpe 1.490、超額總報酬 36.1 個百分點；dynamic GAT-TwoGraph 亦達 51.6\% 總報酬與 1.378 Sharpe。這說明完整中性化架構在長期間能有效把產業與全市場共同影響分離為可排序訊號，但 dynamic graph 並非必然更好；在部分模型中，rolling correlation 提供的即時關係強度可能同時帶來估計噪音，使 IC 改善未必完全轉化為更高 Sharpe。

\section{統一實驗勝出模型與額外彙整}
\label{sec:model_comparison}

表 \ref{tab:results_pred_metrics} 彙整每個 window \(\times\) graph mode 下的勝出模型，屬於前兩節完整指標表的高階導讀。此表同時放入 Test IC、總報酬、Sharpe 與超額報酬最佳模型，因此可以快速辨識「預測層勝出」與「投組層勝出」是否一致。圖 \ref{fig:results_model_comparison_metrics} 則將所有實驗的 Sharpe ratio 視覺化，用來觀察不同 window 與 graph mode 下風險調整後績效的分布。

\input{Sections/tables/results_pred_metrics.tex}

\begin{figure}[H]
  \centering
  \includegraphics[width=\textwidth]{runs_unified/analysis_report/charts/portfolio_sharpe_all_experiments.png}
  \caption{統一實驗中各模型於不同時間窗與圖模式下之 Sharpe ratio。}
  \label{fig:results_model_comparison_metrics}
\end{figure}

表 \ref{tab:results_pred_metrics} 顯示，不存在單一模型在所有評估層面全面占優。Short/static 組合中，GAT-Universe 同時取得最高 Test IC（0.0843）、總報酬（32.13\%）與 Sharpe（7.832）；但 Short/dynamic 組合中，最高 Test IC 仍為 GAT-Universe（0.0812），最高總報酬轉為 DMFM-Full（27.66\%），最高 Sharpe 則為 GAT-TwoGraph（7.318）。Medium/static 組合由 GAT-Industry 取得最高 Test IC（0.0933）與 Sharpe（1.832），但最高總報酬為 GAT-TwoGraph（36.12\%）；Medium/dynamic 組合則由 GAT-Universe 取得最高總報酬（38.31\%）與 Sharpe（1.866）。Long/static 組合中，DMFM-Full 取得最高總報酬（54.32\%）與 Sharpe（1.490），而 Long/dynamic 組合則由 GAT-TwoGraph 取得最高總報酬（51.59\%）與 Sharpe（1.378），GAT-Universe 則有最高 Test IC（0.0955）。

圖 \ref{fig:results_model_comparison_metrics} 的重點在於 Sharpe 的跨組分布。Short window 的 Sharpe 整體偏高，主要受到測試期短且 0050 本身強勢的背景影響，因此不宜將 Short 的 Sharpe 直接外推為長期績效。Medium 與 Long window 的 Sharpe 更能反映模型在多期市場狀態下的穩定性：GAT-Industry、GAT-Universe、GAT-TwoGraph 與 DMFM-Full 多數位於 baseline 上方，但 XGBoost 在多個 static window 仍接近圖模型，顯示本文的貢獻不是簡單宣稱深度圖模型一定優於傳統機器學習，而是指出圖結構與中性化設計在特定市場狀態與投資轉化環節中提供增量。

\section{Dynamic graph 與 Static graph 比較}
\label{sec:dynamic_static_results}

表 \ref{tab:results_dynamic_static_deltas} 比較同一 window 與同一 factor variant 下，dynamic graph 相對 static graph 的 Test IC、總報酬與 Sharpe 差異。這張表的導讀方式應以「delta」而非絕對值為主：正值代表 dynamic graph 改善該指標，負值則代表 dynamic graph 使該指標下降。圖 \ref{fig:results_dynamic_static_deltas} 將相同資訊視覺化，使不同模型與 window 的改善方向更容易比較。

\input{Sections/tables/results_dynamic_static_deltas.tex}

\begin{figure}[H]
  \centering
  \includegraphics[width=\textwidth]{runs_unified/analysis_report/charts/dynamic_minus_static_deltas.png}
  \caption{Dynamic graph 相對 static graph 之 Test IC、總報酬與 Sharpe 差異。}
  \label{fig:results_dynamic_static_deltas}
\end{figure}

由表 \ref{tab:results_dynamic_static_deltas} 可知，dynamic graph 並非單向優勢。18 個可配對比較中，dynamic graph 使 Sharpe 改善 10 組、總報酬改善 11 組，但 Test IC 平均下降 0.0022。分時間窗觀察，Short window 的 dynamic graph 平均 Test IC 下降較明顯，但部分模型仍改善投組表現，例如 DMFM-Full 的總報酬增加 5.28 個百分點、Sharpe 增加 1.367。Medium window 是 dynamic graph 最有利的區間，GAT-Universe 的總報酬增加 10.92 個百分點、Sharpe 增加 0.407，DMFM-Full 的 Test IC 亦增加 0.0093。Long window 則呈現模型分化：GAT-Universe 與 DMFM-Full 的 Test IC 分別增加 0.0135 與 0.0122，但 DMFM-Full 的總報酬與 Sharpe 反而下降，顯示排序相關性的改善不必然等同於 top portfolio 的風險調整績效改善。

圖 \ref{fig:results_dynamic_static_deltas} 進一步呈現 dynamic graph 的異質效果。若 rolling correlation edge 真能穩定捕捉市場關係，應可期待多數模型在 IC 與 Sharpe 上同向改善；但結果顯示改善集中於特定 window 與特定架構，尤其 Medium/dynamic 的 GAT-Universe，而非全面發生。這支持本文將 dynamic graph 視為待檢驗設計，而非預設更佳的假設。rolling correlation edge 能提供即時關係強度，但也會引入窗口估計噪音與期間敏感性；因此，dynamic graph 的價值應依 window、模型結構與投組目標共同判讀。

\section{模型穩定性與投組績效}
\label{sec:portfolio_performance}

表 \ref{tab:results_portfolio_metrics} 從跨 window 與 graph mode 的角度彙整模型穩定性，主要觀察平均 Sharpe、Sharpe 中位數、Sharpe 標準差、平均總報酬與平均 Test IC。這張表補足前面逐組比較的不足：單一 window 的最佳模型可能受特定市場狀態影響，而跨實驗平均與標準差能更直接反映模型在不同環境下的穩定程度。圖 \ref{fig:results_risk_return_scatter} 則以風險報酬散點呈現各策略在報酬與波動之間的位置。

表 \ref{tab:results_portfolio_metrics} 顯示，GAT-Industry 具有最高平均 Sharpe（3.438），代表其在不同 window 與 graph mode 下的風險調整後表現相對穩定；DMFM-Full 則具有最高平均總報酬（36.40\%），顯示完整中性化架構在報酬水準上具競爭力；GAT-Universe 具有最高平均 Test IC（0.0858），說明全市場圖在截面排序相關性上最突出。相較之下，Linear 的平均 Test IC（0.0284）與平均總報酬（14.12\%）皆落後，顯示單純線性映射難以充分吸收高維技術特徵與市場關係資訊。

圖 \ref{fig:results_risk_return_scatter} 的導讀重點是同時觀察報酬與波動，而不是只看右側高報酬點。若某模型位於右上方，代表報酬較高但波動也高；若位於右下方，則代表較有效率的風險報酬轉換。XGBoost 在 static baseline 中仍具競爭力，平均 Sharpe 為 3.225，且在 Short/static 的 Sharpe 達 7.608，接近該組最佳 GAT-Universe 的 7.832。此結果提醒，baseline 並非完全被深度或圖模型取代；結構化 tabular 特徵在短樣本中仍可能由 tree-based 模型有效利用，而圖模型的價值較適合從跨期間穩定性、關係建模與投組層級轉化共同評估。

\section{注意力特徵分析}
\label{sec:feature_importance}

表 \ref{tab:results_dmfm_attention} 彙整 factor attention 在各組實驗中排名前五的特徵，圖 \ref{fig:results_dmfm_attention_top5} 則統計這些前五特徵跨實驗出現的頻率。這組結果的目的不是宣稱注意力權重等同因果效果，而是作為模型偏好訊號的描述性分析：若某些特徵反覆出現在高權重位置，代表模型在不同 window、graph mode 與 variant 下，經常依賴該類技術訊號形成排序分數。

\input{Sections/tables/results_dmfm_attention.tex}

\begin{figure}[H]
  \centering
  \includegraphics[width=0.9\textwidth]{runs_unified/analysis_report/charts/attention_feature_frequency.png}
  \caption{注意力前五特徵之跨實驗出現頻率。}
  \label{fig:results_dmfm_attention_top5}
\end{figure}

表 \ref{tab:results_dmfm_attention} 與圖 \ref{fig:results_dmfm_attention_top5} 顯示，注意力前五特徵最常落在 rolling min/max、波動與量能相關欄位。出現次數最高的是 \texttt{roll\_min\_5} 與 \texttt{roll\_min\_20}（皆為 12 次），其次為 \texttt{roll\_min\_60}（11 次），並包含 \texttt{std\_ret\_10}、\texttt{roll\_max\_5}、\texttt{roll\_max\_10}、\texttt{up\_vol\_20} 與 \texttt{vol\_over\_ma\_20} 等特徵。這表示 factor variants 多半透過近期價格區間、低點位置、波動狀態與量能變化進行截面排序，而非依賴單一估值或單一動能訊號。

結合前述投組結果可知，模型優勢並非來自某一固定特徵，而是由 encoder、圖聚合、中性化與 decoder 將多類特徵重新組合後形成排序分數。注意力分析因此適合作為「模型使用哪些訊號」的補充證據，但不應單獨用來判定特徵的經濟因果性。完整全模型關鍵指標與分析圖表整理於附錄 \ref{sec:appendix_extra_outputs}。
