\chapter{實驗結果}
\label{ch:results}

本章根據 \texttt{runs\_unified/analysis\_report} 的統一彙整結果，回答本文的核心研究問題：在相同 TEJ 日頻資料、相同時間切分與相同回測設定下，圖注意力、多因子表示、分層中性化與 dynamic graph 是否能提升台股截面選股訊號。完整實驗共 45 組，包含 static / dynamic 兩種圖模式、Short / Medium / Long 三種樣本視窗、三個非圖 baseline，以及六個 factor ablation models。需特別說明，baseline 不使用圖結構，因此預設僅列於 static graph 口徑；dynamic graph 的比較主要適用於六個 factor variants。

表 \ref{tab:results_pred_metrics} 彙整每個 window \(\times\) graph mode 下的勝出模型。結果顯示，不存在單一模型在所有評估層面全面占優。Short/static 組合中，GAT-Universe 同時取得最高 Test IC（0.0843）、總報酬（32.13\%）與 Sharpe（7.832）；但 Short/dynamic 組合中，最高 Test IC 仍為 GAT-Universe（0.0812），最高總報酬轉為 DMFM-Full（27.66\%），最高 Sharpe 則為 GAT-TwoGraph（7.318）。Medium/static 組合由 GAT-Industry 取得最高 Test IC（0.0933）與 Sharpe（1.832），但最高總報酬為 GAT-TwoGraph（36.12\%）；Medium/dynamic 組合則由 GAT-Universe 取得最高總報酬（38.31\%）與 Sharpe（1.866）。Long/static 組合中，DMFM-Full 取得最高總報酬（54.32\%）與 Sharpe（1.490），而 Long/dynamic 組合則由 GAT-TwoGraph 取得最高總報酬（51.59\%）與 Sharpe（1.378），GAT-Universe 則有最高 Test IC（0.0955）。

上述結果說明，預測層與投組層的最優模型並不必然一致。Test IC 代表截面排序與去均值標籤的相關性；Sharpe 與總報酬則取決於 top 10\% 選股、5 日再平衡與原始 forward return 的實際轉化效果。因此，本文不以單一 regression loss 或單一 IC 判斷模型優劣，而是將預測層與投組層分開解讀。

\section{Dynamic graph 與 Static graph 比較}
\label{sec:dynamic_static_results}

\input{Sections/tables/results_dynamic_static_deltas.tex}

\begin{figure}[H]
  \centering
  \includegraphics[width=\textwidth]{runs_unified/analysis_report/charts/dynamic_minus_static_deltas.png}
  \caption{Dynamic graph 相對 static graph 之 Test IC、總報酬與 Sharpe 差異。}
  \label{fig:results_dynamic_static_deltas}
\end{figure}

由表 \ref{tab:results_dynamic_static_deltas} 可知，dynamic graph 並非單向優勢。18 個可配對比較中，dynamic graph 使 Sharpe 改善 10 組、總報酬改善 11 組，但 Test IC 平均下降 0.0022。分時間窗觀察，Short window 的 dynamic graph 平均 Test IC 下降較明顯，但部分模型仍改善投組表現，例如 DMFM-Full 的總報酬增加 5.28 個百分點、Sharpe 增加 1.367。Medium window 是 dynamic graph 最有利的區間，GAT-Universe 的總報酬增加 10.92 個百分點、Sharpe 增加 0.407，DMFM-Full 的 Test IC 亦增加 0.0093。Long window 則呈現模型分化：GAT-Universe 與 DMFM-Full 的 Test IC 分別增加 0.0135 與 0.0122，但 DMFM-Full 的總報酬與 Sharpe 反而下降，顯示排序相關性的改善不必然等同於 top portfolio 的風險調整績效改善。

這個結果支持本文將 dynamic graph 視為待檢驗設計，而非預設更佳的假設。rolling correlation edge 能提供即時關係強度，但也會引入窗口估計噪音與期間敏感性；因此，dynamic graph 的價值應依 window、模型結構與投組目標共同判讀。

\label{sec:portfolio_performance}

表 \ref{tab:results_portfolio_metrics} 從跨實驗穩定性角度比較模型。GAT-Industry 具有最高平均 Sharpe（3.438），代表其在不同 window 與 graph mode 下的風險調整後表現相對穩定；DMFM-Full 則具有最高平均總報酬（36.40\%），顯示完整中性化架構在報酬水準上具競爭力；GAT-Universe 具有最高平均 Test IC（0.0858），說明全市場圖在截面排序相關性上最突出。相較之下，Linear 的平均 Test IC（0.0284）與平均總報酬（14.12\%）皆落後，顯示單純線性映射難以充分吸收高維技術特徵與市場關係資訊。

Baseline 並非完全被深度或圖模型取代。XGBoost 在 static baseline 中仍具競爭力，平均 Sharpe 為 3.225，且在 Short/static 的 Sharpe 達 7.608，接近該組最佳 GAT-Universe 的 7.832。此結果提醒，結構化 tabular 特徵在短樣本中仍可能由 tree-based 模型有效利用；圖模型的價值較適合從跨期間穩定性、關係建模與投組層級轉化共同評估。

表 \ref{tab:results_dmfm_attention} 與圖 \ref{fig:results_dmfm_attention_top5} 顯示，注意力前五特徵最常落在 rolling min/max、波動與量能相關欄位。出現次數最高的是 \texttt{roll\_min\_5} 與 \texttt{roll\_min\_20}（皆為 12 次），其次為 \texttt{roll\_min\_60}（11 次），並包含 \texttt{std\_ret\_10}、\texttt{roll\_max\_5}、\texttt{roll\_max\_10}、\texttt{up\_vol\_20} 與 \texttt{vol\_over\_ma\_20} 等特徵。這表示 factor variants 多半透過近期價格區間、低點位置、波動狀態與量能變化進行截面排序，而非依賴單一估值或單一動能訊號。

注意力結果應解讀為模型偏好的訊號配置，而非因果效果估計。結合前述投組表現可知，模型優勢並非來自某一固定特徵，而是由 encoder、圖聚合、中性化與 decoder 將多類特徵重新組合後形成排序分數。完整全模型關鍵指標與分析圖表整理於附錄 \ref{sec:appendix_extra_outputs}。
