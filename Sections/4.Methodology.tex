\chapter{實驗結果}
\label{ch:results}

本章根據正式實驗結果，系統性地回答本文的核心研究問題：在台灣股票市場橫截面報酬預測任務中，模型結構、圖拓樸設計與訓練視窗長度如何共同影響截面排序訊號品質與投資組合表現。完整實驗共 27 組，涵蓋短期、中期、長期三種訓練視窗，以及線性迴歸、XGBoost、LSTM、DNN、GAT-industry、GAT-universe、GAT-TwoGraph、GAT-IndNeutral 與 GAT-full 九類模型。非圖基準模型不使用圖結構；GAT 拓樸系列與 GAT 中性化系列均採用相同封存資料與圖結構，以確保比較結果不受資料切分差異或圖重估方式干擾。

本章以 Daily IC 與 ICIR 作為主要評估指標，因本研究任務為橫截面排序而非單純數值迴歸，排序訊號的跨期穩健性比均方誤差更直接反映模型在截面投資策略中的可用性。投資組合績效則用以驗證排序訊號是否能轉化為前 10\% 等權做多策略的實質報酬。各節依序以非圖基準模型、圖拓樸比較與中性化機制為主軸展開分析，最後以訓練視窗長度的橫向比較統整各類模型的跨期表現。

\begin{enumerate}

  \item \textbf{圖拓樸設計是必要非充分條件。}GAT-universe（全市場圖）在長期視窗的 ICIR 僅 0.526，低於 GAT-industry（0.677）與 DNN（0.726），說明圖若不搭配具有經濟意義的拓樸，將因過度平滑化降低截面辨識力。統計檢定進一步確認：僅中性化延伸模型（GAT-IndNeutral、GAT-full）能在 Diebold-Mariano 配對檢定中顯著優於 DNN（詳見第~\ref{subsec:graph_contribution}~節）。

  \item \textbf{GAT 中性化系列在長期視窗多指標同時領先。}長期視窗 GAT-IndNeutral 取得最高 ICIR（0.786），GAT-full 取得最高 Sharpe 比率（1.849）；兩者的優勢隨訓練資料充足而持續擴大。

  \item \textbf{訓練視窗長度影響排序訊號轉化為投資組合績效的程度。}短期視窗（116 天）下九個模型的 Sharpe 比率介於 $-0.423$ 至 $+0.057$，測試總報酬均為負；中、長期視窗 Sharpe 整體轉正，長期最佳達 1.849。此結果顯示，較長訓練視窗所涵蓋的市場情境，有助於排序訊號在測試期間轉化為較穩定的投資組合績效。
\end{enumerate}

\section{非圖基準模型之實證分析}
\label{sec:baseline_results}
% =========================================================

本節依序分析線性迴歸、XGBoost、LSTM 與 DNN 四類非圖基準模型。此四類模型均不使用圖結構，其結果一方面作為後續圖注意力模型的比較基準，另一方面直接揭示特徵工程在截面排序任務中的有效程度。

\begin{figure}[H]
  \centering
  \includegraphics[width=\textwidth]{img/ch4_fig_baseline_comparison.png}
  \caption{非圖基準模型於三個訓練視窗所對應之測試視窗 Daily IC（左）與 Sharpe 比率（右）。黃色為短期、綠色為中期、藍色為長期測試視窗。}
  \label{fig:ch4_baseline_comparison}
\end{figure}

圖~\ref{fig:ch4_baseline_comparison} 顯示四類非圖基準模型於三個訓練視窗所對應測試視窗之 Daily IC 與 Sharpe 比率。短期測試視窗中，僅 XGBoost 的 Sharpe 比率略高於零；中期與長期測試視窗中，四類模型的 Sharpe 比率均轉為正值。DNN 的 Daily IC 在短期與長期測試視窗中為非圖基準模型最高，中期測試視窗則以 XGBoost 最高。

\subsection{線性迴歸：線性擬合的侷限}
\label{subsec:linear_results}

線性迴歸在本文所有模型中整體表現最弱。三個訓練視窗的平均 Daily IC 僅 0.0168，短期視窗 Sharpe 比率為 -0.423，顯示 56 維技術與估值特徵之間存在難以透過線性擬合捕捉的非線性交互作用。即便在訓練樣本最充足的長期視窗下，線性迴歸的截面排序能力仍明顯落後於其他非圖模型。此結果確認橫截面報酬預測任務的非線性本質，印證以非線性模型作為主要實驗架構的必要性。

\input{Sections/tables/results_linear_detail.tex}

表~\ref{tab:results_linear_detail} 顯示，Linear 的測試 Daily IC 於中期與長期視窗均高於短期視窗，Sharpe 比率亦由短期視窗的負值轉為中期與長期視窗的正值；然而，其測試 Daily IC 與測試 ICIR 仍低於其他非圖基準模型。

\subsection{XGBoost：梯度提升模型之過擬合診斷}
\label{subsec:xgboost_results}

XGBoost 在長期視窗的投資組合總報酬達 143.08\%，為本文所有模型中最高。然而，表 \ref{tab:results_xgboost_gap} 顯示，其長期視窗訓練 Daily IC 為 0.2570，測試 Daily IC 僅為 0.0408，Daily IC 測試/訓練比為 0.16，顯示在投資組合層級的高報酬背後隱含顯著的過擬合風險。

\input{Sections/tables/results_xgboost_gap.tex}

表~\ref{tab:results_xgboost_gap} 亦顯示，XGBoost 在三個訓練視窗的 Daily IC 測試/訓練比均偏低，其中短期視窗僅為 0.06，長期視窗亦僅為 0.16。本文將 XGBoost 定位為梯度提升模型的代表性基準，但其訊號穩健性需一併揭露，不宜以單一總報酬指標作為模型品質的最終評判依據。

\subsection{LSTM：時間序列記憶的邊際效益}
\label{subsec:lstm_results}

LSTM 在長期視窗的 Sharpe 比率達 1.336，較線性迴歸有所改善，顯示個股時間序列的歷史記憶機制具有一定程度的排序效益。然而，LSTM 在三個視窗的 Daily IC 與 Sharpe 比率均低於 DNN、GAT 拓樸系列與 GAT 中性化系列，顯示以逐股時序方式建模在截面排序任務中的邊際貢獻有限。

\input{Sections/tables/results_lstm_detail.tex}

表~\ref{tab:results_lstm_detail} 顯示，LSTM 的測試 Daily IC、測試 ICIR、測試 Sharpe 與測試總報酬均隨訓練視窗延長而改善；其中長期視窗測試 Daily IC 為 0.0418，Sharpe 比率為 1.336。

截面排序的核心在於比較同一時點不同股票的相對強弱，而 LSTM 以逐股時序方式獨立建模，無法直接利用截面內各股票之間的關係資訊。相對地，DNN 透過跨股截面批次正規化與全連接層，較 LSTM 更適合從多股共同特徵中萃取截面排序訊號，此點亦在下一小節得到驗證。

\subsection{DNN：非圖模型的截面排序效益}
\label{subsec:dnn_results}

DNN 是本文非圖基準模型中最具參考價值的對照組。儘管不使用任何圖結構，DNN 在短期視窗取得全模型最高 Daily IC（0.0396）與 ICIR（0.408）；在中期視窗取得最高 Sharpe 比率（0.980）與最高總報酬（39.13\%）；在長期視窗亦達 Daily IC 0.0535 與 Sharpe 比率 1.564，與 GAT 拓樸系列及 GAT 中性化系列具有競爭性。

\input{Sections/tables/results_dnn_detail.tex}

表~\ref{tab:results_dnn_detail} 顯示，DNN 的測試 Daily IC 與測試 ICIR 由短期至長期視窗持續提高；短期視窗雖具有較高的測試 Daily IC，測試 Sharpe 與測試總報酬仍為負值，中期與長期視窗則轉為正值。

此結果具有重要的方法論意義：56 維技術與估值特徵搭配截面標準化的特徵工程設計，已在本資料集中提供相當程度的截面排序能力。後續第 \ref{sec:topology_results} 節與第 \ref{sec:neutralization_results} 節所討論的圖注意力機制效益，均應以 DNN 作為非圖對照基準進行評估，而非以更弱的線性或 LSTM 基準作為比較起點。

\section{圖拓樸結構之排序效益比較}
\label{sec:topology_results}
% =========================================================

本節在建立非圖基準後，進一步比較不同圖拓樸設計的截面排序效益。分析以長期視窗為主要討論情境，因長期視窗的訓練樣本最充足，能較清楚地區分圖拓樸設計的差異效果；中期視窗結果作為輔助參照。

\subsection{產業圖與全市場圖之對比}
\label{subsec:industry_vs_universe}

表 \ref{tab:results_long_topology_train_test} 聚焦比較 DNN、GAT-industry 與 GAT-universe。訓練 Daily IC 反映樣本內擬合能力，測試 Daily IC 與測試 ICIR 則直接衡量樣本外排序訊號品質。最直接的觀察是，GAT-universe 的測試 Daily IC（0.0452）與測試 Sharpe（1.506）均低於不使用圖結構的 DNN（測試 Daily IC 0.0535，測試 Sharpe 1.564）。這表示全截面完全連結並不必然提供有效的增量資訊：當每一檔股票都與全市場所有股票交換訊息時，模型容易過度吸收市場共同成分，反而削弱個股截面差異的辨識能力，即文獻所指之過度平滑化效應。

\input{Sections/tables/results_long_topology_train_test.tex}

相對地，GAT-industry 在長期視窗的 Daily IC 為 0.0543、Sharpe 比率為 1.779，均優於 GAT-universe 與 DNN。中期視窗的結果方向一致：GAT-industry 的 Daily IC（0.0445）與 Sharpe 比率（0.900）均高於 GAT-universe（0.0377、0.755）。此結果支持具有經濟意義之局部圖拓樸的有效性：產業圖提供相對穩定的同業關係連結，使模型能在同產業股票之間進行相對強弱比較，而不引入與個股截面差異無關的全市場共同波動。

\subsection{雙圖架構（GAT-TwoGraph）之效益分析}
\label{subsec:twograph_results}

GAT-TwoGraph 同時融合產業圖與全市場圖的訊息傳遞，其結果呈現截面排序訊號品質與投資組合風險報酬之間的取捨。長期視窗下，GAT-TwoGraph 的 Daily IC 為 0.0512，低於 GAT-industry 的 0.0543，但 Sharpe 比率提升至 1.827。

\input{Sections/tables/results_twograph_detail.tex}

表~\ref{tab:results_twograph_detail} 顯示，中期視窗下 GAT-TwoGraph 的測試 Daily IC 與測試 ICIR 均高於 GAT-industry，但測試 Sharpe 與測試總報酬較低；長期視窗下 GAT-TwoGraph 的測試 Sharpe 與測試總報酬較高，但測試 Daily IC 與測試 ICIR 較低。此結果說明，雙圖架構在長期視窗下可改善投資組合的風險報酬特徵，但平均截面排序訊號品質未必同步提升。圖拓樸的選擇應視評估目標而定，以 ICIR 衡量排序訊號穩健性或以 Sharpe 比率衡量投資組合效益，兩者的最佳拓樸選擇未必相同。

\subsection{統計穩健性檢定}
\label{subsec:statistical_tests}

為驗證上述結果的統計穩健性，本文針對長期視窗逐日 IC 序列進行 Newey-West HAC $t$ 檢定、Block Bootstrap ICIR 信賴區間估計與 Diebold-Mariano 配對比較。各檢定之虛無假說與對立假說如下：

\begin{align}
H_0^{\mathrm{NW}} &: \mathbb{E}[\mathrm{IC}_t] = 0,
&
H_1^{\mathrm{NW}} &: \mathbb{E}[\mathrm{IC}_t] > 0, \\
H_0^{\mathrm{BB}} &: \mathrm{ICIR} \leq 0,
&
H_1^{\mathrm{BB}} &: \mathrm{ICIR} > 0, \\
H_0^{\mathrm{DM}} &: \mathbb{E}[d_t] = 0,
&
H_1^{\mathrm{DM}} &: \mathbb{E}[d_t] > 0,
\end{align}
其中 $d_t=\mathrm{IC}_{A,t}-\mathrm{IC}_{\mathrm{DNN},t}$，模型 $A$ 為各 GAT 模型。Newey-West HAC $t$ 檢定與 Diebold-Mariano 配對比較皆採單尾檢定；Block Bootstrap 則以 95\% 信賴區間下界是否大於零作為判定依據。

\textbf{Newey-West HAC $t$ 檢定。}截面 IC 序列並非獨立同分布（i.i.d.）。由於同一批股票在相鄰交易日之間共享部分資訊環境，IC 序列存在顯著的序列自相關（長期視窗一階自相關約 0.53）。本文依 Andrews（1991）~\cite{andrews1991autocovariance}自動選階規則設定 lag $= 5$，並以 Newey-West 異質性-自相關一致估計量~\cite{newey1987simple}修正標準誤。

\textbf{Block Bootstrap ICIR 信賴區間。}ICIR 為 IC 均值除以 IC 標準差，其抽樣分布在有序列相關的資料中不服從標準常態。本文採用環形 Block Bootstrap（block size $= 10$，重複 $B = 2000$ 次），在每次重抽樣中保留序列的局部時間依存結構，並計算 95\% 百分位信賴區間。

\textbf{Diebold-Mariano 配對比較。}本文以 DNN 為非圖基準，檢驗各 GAT 模型之逐日 IC 是否顯著高於 DNN，並以 Newey-West HAC 估計量修正 $d_t$ 序列之標準誤。

\input{Sections/tables/results_stat_tests.tex}

表~\ref{tab:results_stat_tests} 呈現全部 GAT 模型與 DNN 的三類檢定完整結果，可從三個層次解讀：

\textbf{（一）平均 IC 顯著異於零。}Newey-West HAC $t$ 檢定顯示，全部五個 GAT 模型與 DNN 之長期視窗平均 IC 均在 $p < 0.001$ 水準下顯著大於零，$t$ 統計量介於 6.879（GAT-TwoGraph）至 11.759（GAT-IndNeutral）之間。此結果確認各模型排序訊號均非源於噪音，但不能區分模型間的相對優劣。

\textbf{（二）ICIR 信賴區間下界均大於零。}Block Bootstrap 95\% 信賴區間顯示，全部模型之 ICIR 信賴區間下界均遠大於零，GAT-IndNeutral 的 CI 為 $[0.624,\ 0.957]$，GAT-full 為 $[0.574,\ 0.909]$，DNN 為 $[0.563,\ 0.901]$。ICIR 較低的模型（GAT-TwoGraph $[0.365,\ 0.663]$、GAT-universe $[0.395,\ 0.667]$）信賴區間仍完全大於零，確認所有模型之跨期排序一致性本身具統計意義。

\textbf{（三）DM 配對比較呈現明確分層。}以 DNN 為非圖基準，GAT-IndNeutral（DM $= 2.330$，$p = 0.010$）與 GAT-full（$p = 0.031$）對 DNN 的每日 IC 差異在 5\% 顯著水準下可統計辨識；GAT-industry、GAT-universe 與 GAT-TwoGraph 均未達顯著水準（$p > 0.05$），GAT-universe 的 DM $= -1.983$ 更顯示其 IC 在點估計上低於 DNN。此分層揭示：(A)(B) 確認所有模型訊號均真實存在，(C) 則進一步說明\textbf{中性化機制——而非圖連結本身——才是圖模型相對 DNN 可統計驗證的增量來源}。

表~\ref{tab:results_portfolio_metrics} 以模型類別為橫向切面，呈現各訓練視窗所對應測試集的平均指標。短期視窗的訓練樣本最為有限，多數模型雖已具正向 Daily IC，但測試總報酬均為負。DNN 在短期視窗的 Daily IC 為 0.0396，Sharpe 比率卻為 $-0.098$；GAT 中性化系列的平均 Daily IC 為 0.0278，平均 Sharpe 比率僅 $-0.084$。短期視窗下弱正向的排序訊號，尚不足以在扣除交易成本前穩定轉化為可投資報酬。

中期視窗擴大訓練樣本後，截面排序訊號品質與投資組合績效均改善，惟兩者的模型排名並非完全一致。就截面排序訊號而言，GAT 中性化系列平均 Daily IC 提升至 0.0548，GAT-full 取得中期最高 Daily IC（0.0596）；就投資組合績效而言，DNN 取得最高 Sharpe 比率（0.980）與最高總報酬（39.13\%）。此結果說明，較高的 IC 並不自動轉化為較高的投資組合 Sharpe，IC 反映截面排序能力，Sharpe 則涵蓋排序訊號轉化為持倉集中度與報酬的效率，兩者應分層解讀。

長期視窗的改善最為穩定，且訊號品質與投資組合績效均呈現一致的改善趨勢。就截面排序訊號，GAT 中性化系列平均 Daily IC 為 0.0587、平均 ICIR 為 0.760，居各類別最高；DNN 平均 Daily IC 為 0.0535；GAT 拓樸系列為 0.0502。就投資組合績效，GAT 中性化系列平均 Sharpe 比率達 1.826，GAT 拓樸系列為 1.704，DNN 為 1.564。長期視窗的訓練期橫跨多種市場情境，使截面排序關係在樣本外更具泛化能力。

短期視窗中，DNN 在 Daily IC（0.0396）與 ICIR（0.408）上最高，但最佳 Sharpe 比率僅 GAT-industry 的 0.057，且所有模型總報酬均為負；中期視窗中，GAT-full 取得最高 Daily IC（0.0596）與 ICIR（0.628），DNN 取得最高 Sharpe 比率（0.980）與總報酬（39.13\%）；長期視窗中，GAT-IndNeutral 取得最高 Daily IC（0.0590）與 ICIR（0.786），GAT-full 取得最高 Sharpe 比率（1.849），XGBoost 取得最高總報酬（143.08\%，但伴隨顯著過擬合風險，詳見第 \ref{subsec:xgboost_results} 節）。

此結果揭示一項方法論上的重要考量：若僅以短期視窗進行模型比較，易低估圖注意力模型與 GAT 中性化系列的潛在效益；若僅觀察長期視窗的投資組合報酬，則因 XGBoost 的過擬合情形高估其排序穩健性。較完整的評估方式應同時報告三個訓練視窗，並以 IC/ICIR 與投資組合績效共同衡量。

\subsection{評估指標選擇之方法論考量}
\label{subsec:metric_discussion}

本實驗顯示，均方根誤差（RMSE）不適合作為跨架構的唯一比較指標。以長期視窗為例，線性迴歸、LSTM 與 GAT-industry 的 RMSE 約為 0.0484 至 0.0485，而 DNN、GAT-IndNeutral 與 GAT-full 的 RMSE 分別為 0.1336、0.1160 與 0.1817；然而後三者在 Daily IC、ICIR 與 Sharpe 比率上均明顯優於前者。RMSE 偏高的原因在於因子注意力與中性化模型的輸出尺度與原始報酬空間不完全一致，且截面排序任務所需者為相對強弱排序而非精確數值預測。此結果印證以 IC/ICIR 為主要評估指標的方法論選擇：排序訊號的跨期穩健性比均方誤差更直接反映模型在橫截面投資策略中的可用性。各模型之完整預測品質指標、投資組合績效與全模型關鍵指標，請分別參閱附錄表~\ref{tab:results_prediction_quality_full}、表~\ref{tab:results_portfolio_0050_full} 與表~\ref{tab:appendix_unified_key_metrics}。

\subsection{圖模型增量貢獻之定位}
\label{subsec:graph_contribution}

綜合本章各節分析，圖注意力機制的增量貢獻可從三個層次定位，每個層次均有對應的實證與統計依據支撐：

\textbf{第一，截面排序訊號品質。}GAT 中性化系列在長期視窗的測試 ICIR 最高（GAT-IndNeutral 0.786、GAT-full 0.735），優於純圖模型（0.504--0.677）與 DNN（0.726）。此優勢隨訓練資料充足而擴大，在短期視窗則收窄甚至消失，顯示圖模型的訊號品質增益取決於訓練資料規模。

\textbf{第二，統計可辨識性。}第~\ref{subsec:statistical_tests}~節的 Diebold-Mariano 配對比較顯示，長期視窗中僅 GAT-IndNeutral（$p = 0.010$）與 GAT-full（$p = 0.031$）對 DNN 達顯著水準；GAT-industry、GAT-universe 與 GAT-TwoGraph 雖在點估計上接近 DNN，統計上無法與之區分。此結果直接說明：\textbf{中性化機制，而非圖連結本身，是圖模型相對 DNN 可統計驗證的增量貢獻}——圖拓樸的訊號精煉效果，必須透過中性化約束去除產業與市場共同因子後才能顯現。

\textbf{第三，泛化能力。}表~\ref{tab:results_overview_master}「Daily IC 測試/訓練比」欄位顯示，長期視窗中 GAT-IndNeutral（0.38）與 DNN（0.38）並列最高，在取得更高測試 Daily IC 的同時維持相近的泛化效率。XGBoost 長期比值（0.16）仍顯著偏低，其高報酬需合併過擬合診斷解讀。

\subsection{研究限制}
\label{subsec:limitations}

本實驗存在若干重要限制，應於解讀結果時一併考慮。

其一，回測尚未納入交易成本、市場衝擊與換手率約束，現有績效數字應視為無摩擦假設下的上限估計，實際可落地的超額報酬將低於本文所報告的數值。

其二，本文已針對長期視窗 GAT-IndNeutral 與 DNN 進行 Newey-West $t$ 檢定（$t = 11.76$，$p < 0.001$）與 Diebold-Mariano 配對比較（$p = 0.010$），以及 Block Bootstrap ICIR 信賴區間估算（95\% CI = [0.624, 0.957]）。然而，短期與中期視窗及其餘模型組合的逐一統計檢定、分年度情境分析與跨市場條件分組驗證，有待後續研究進一步補充，以更全面評估各模型在不同市場環境下的跨期一致性。

其三，本文圖結構所使用之產業分類為穩定外生資訊，不涉及前視偏誤；但若未來研究採用動態圖（如滾動相關係數）構建邊，則需嚴格確保圖建構僅使用截至當前交易日的訓練期資訊，以避免前視偏誤影響評估結果。
