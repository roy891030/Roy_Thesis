\chapter{實驗結果}
\label{ch:results}

本章根據正式實驗結果，系統性地回答本文的核心研究問題：在台灣股票市場橫斷面報酬預測任務中，模型結構、圖拓樸設計與訓練視窗長度如何共同影響截面排序訊號品質與投資組合表現。完整實驗共 27 組，涵蓋短期、中期、長期三種訓練視窗，以及線性迴歸、XGBoost、LSTM、DNN、GAT-industry、GAT-universe、GAT-TwoGraph、GAT-IndNeutral 與 GAT-full 九類模型。非圖基準模型不使用圖結構；GAT 系列與中性化延伸模型均採用相同封存資料與圖結構，以確保比較結果不受資料切分差異或圖重估方式干擾。

本章以 Daily IC 與 ICIR 作為主要評估指標，因本研究任務為橫斷面排序而非單純數值迴歸，排序訊號的跨期穩健性比均方誤差更直接反映模型在截面投資策略中的可用性。投資組合績效則用以驗證排序訊號是否能轉化為前 10\% 等權做多策略的實質報酬。各節依序以非圖基準模型、圖拓樸比較與中性化機制為主軸展開分析，最後以訓練視窗長度的橫向比較統整各類模型的跨期表現。

  \item \textbf{圖拓樸設計是必要非充分條件。}GAT-universe（全市場圖）在長期視窗的 ICIR 僅 0.526，低於 GAT-industry（0.677）與 DNN（0.726），說明圖若不搭配具有經濟意義的拓樸，將因過度平滑化降低截面辨識力。統計檢定進一步確認：僅中性化延伸模型（GAT-IndNeutral、GAT-full）能在 Diebold-Mariano 配對檢定中顯著優於 DNN（詳見第~\ref{subsec:graph_contribution}~節）。

  \item \textbf{GAT 中性化系列在長期視窗多指標同時領先。}長期視窗 GAT-IndNeutral 取得最高 ICIR（0.786），GAT-full 取得最高 Sharpe 比率（1.849）；兩者的優勢隨訓練資料充足而持續擴大。

  \item \textbf{訓練視窗長度是可投資性的前提閾值。}短期視窗（116 天）下九個模型的 Sharpe 比率介於 $-0.423$ 至 $+0.057$，年化報酬普遍為負；中、長期視窗 Sharpe 整體轉正，長期最佳達 1.849。訓練資料規模決定排序訊號能否轉化為可投資報酬，優先於架構設計本身。
\end{enumerate}

\section{非圖基準模型之實證分析}
\label{sec:baseline_results}
% =========================================================

本節依序分析線性迴歸、XGBoost、LSTM 與 DNN 四類非圖基準模型。此四類模型均不使用圖結構，其結果一方面作為後續圖注意力模型的比較基準，另一方面直接揭示特徵工程在截面排序任務中的有效程度。

\subsection{線性迴歸：線性擬合的侷限}
\label{subsec:linear_results}

線性迴歸在本文所有模型中整體表現最弱。三個訓練視窗的平均 Daily IC 僅 0.0168，短期視窗 Sharpe 比率為 -0.423，顯示 56 維技術與估值特徵之間存在難以透過線性擬合捕捉的非線性交互作用。即便在訓練樣本最充足的長期視窗下，線性迴歸的截面排序能力仍明顯落後於其他非圖模型。此結果確認橫斷面報酬預測任務的非線性本質，印證以非線性模型作為主要實驗架構的必要性。

\subsection{XGBoost：梯度提升模型之過擬合診斷}
\label{subsec:xgboost_results}

XGBoost 在長期視窗的投資組合總報酬達 143.08\%，為本文所有模型中最高。然而，表 \ref{tab:results_xgboost_gap} 揭示，其訓練集資訊係數（0.7824）與測試集資訊係數（0.0205）之間存在高達 38.1 倍的差距，顯示在投資組合層級的高報酬背後隱含顯著的過擬合風險。

\input{Sections/tables/results_xgboost_gap.tex}

此訓練-測試差距提示，XGBoost 的長期視窗高報酬源自訓練集的記憶效應，而非具有跨期穩健性的截面排序訊號。本文將 XGBoost 定位為梯度提升模型的代表性基準，但其訊號穩健性需一併揭露，不宜以單一總報酬指標作為模型品質的最終評判依據。

  \label{fig:ch4_xgboost_gap}
\end{figure}

\subsection{LSTM：時間序列記憶的邊際效益}
\label{subsec:lstm_results}

LSTM 在長期視窗的 Sharpe 比率達 1.336，較線性迴歸有所改善，顯示個股時間序列的歷史記憶機制具有一定程度的排序效益。然而，LSTM 在三個視窗的 Daily IC 與 Sharpe 比率均低於 DNN 及後續 GAT 系列，顯示以逐股時序方式建模在截面排序任務中的邊際貢獻有限。

截面排序的核心在於比較同一時點不同股票的相對強弱，而 LSTM 以逐股時序方式獨立建模，無法直接利用截面內各股票之間的關係資訊。相對地，DNN 透過跨股截面批次正規化與全連接層，較 LSTM 更適合從多股共同特徵中萃取截面排序訊號，此點亦在下一小節得到驗證。

\subsection{DNN：非圖模型的截面排序效益}
\label{subsec:dnn_results}

DNN 是本文非圖基準模型中最具參考價值的對照組。儘管不使用任何圖結構，DNN 在短期視窗取得全模型最高 Daily IC（0.0396）與 ICIR（0.408）；在中期視窗取得最高 Sharpe 比率（0.980）與最高總報酬（39.13\%）；在長期視窗亦達 Daily IC 0.0535 與 Sharpe 比率 1.564，與 GAT 系列具有競爭性。

此結果具有重要的方法論意義：56 維技術與估值特徵搭配截面標準化的特徵工程設計，已在本資料集中提供相當程度的截面排序能力。後續第 \ref{sec:topology_results} 節與第 \ref{sec:neutralization_results} 節所討論的圖注意力機制效益，均應以 DNN 作為非圖對照基準進行評估，而非以更弱的線性或 LSTM 基準作為比較起點。

\section{圖拓樸結構之排序效益比較}
\label{sec:topology_results}
% =========================================================

本節在建立非圖基準後，進一步比較不同圖拓樸設計的截面排序效益。分析以長期視窗為主要討論情境，因長期視窗的訓練樣本最充足，能較清楚地區分圖拓樸設計的差異效果；中期視窗結果作為輔助參照。

\subsection{產業圖與全市場圖之對比}
\label{subsec:industry_vs_universe}

表 \ref{tab:results_static_long_topology} 彙整長期視窗全部九類模型的訓練集與測試集核心指標，訓練 IC / 訓練 ICIR 反映樣本內擬合能力，測試 IC / 測試 ICIR 則直接衡量樣本外排序訊號品質。最直接的觀察是，GAT-universe 的測試 IC（0.0452）與 Sharpe 比率（1.506）均低於不使用圖結構的 DNN（測試 IC 0.0535，Sharpe 比率 1.564）。這表示全截面完全連結並不必然提供有效的增量資訊：當每一檔股票都與全市場所有股票交換訊息時，模型容易過度吸收市場共同成分，反而削弱個股截面差異的辨識能力，即文獻所指之過度平滑化效應。

\subsection{雙圖架構（GAT-TwoGraph）之效益分析}
\label{subsec:twograph_results}

GAT-TwoGraph 同時融合產業圖與全市場圖的訊息傳遞，其結果呈現截面排序訊號品質與投資組合風險報酬之間的取捨。長期視窗下，GAT-TwoGraph 的 Daily IC 為 0.0512，低於 GAT-industry 的 0.0543，但 Sharpe 比率提升至 1.827，最大跌幅為 -19.23\%，是同視窗 GAT 系列中最低。

此結果說明，雙圖架構在長期視窗下可改善投資組合的風險報酬特徵，但平均截面排序訊號品質未必同步提升。全市場圖提供的共同因子資訊或有助於在特定市場情境下降低投資組合的系統性風險暴露，但同時亦稀釋產業圖所提供的局部排序訊號。圖拓樸的選擇應視評估目標而定，以 ICIR 衡量排序訊號穩健性或以 Sharpe 比率衡量投資組合效益，兩者的最佳拓樸選擇未必相同。

\section{中性化機制分析}
\label{sec:neutralization_results}
% =========================================================

本節分析 GAT-IndNeutral 與 GAT-full 兩種加入中性化與因子注意力機制的延伸模型，並探討注意力特徵在不同訓練視窗下的分布規律。

\subsection{GAT-IndNeutral 與 GAT-full 之比較}
\label{subsec:indneutral_vs_full}

GAT 中性化類模型在中期與長期視窗的 ICIR 表現最為突出。中期視窗中，GAT-full 取得全模型最高 Daily IC（0.0596）與 ICIR（0.628）；長期視窗中，GAT-IndNeutral 取得最高 Daily IC（0.0590）與 ICIR（0.786），GAT-full 亦有 Daily IC 0.0583 與 Sharpe 比率 1.849。以 ICIR 作為衡量排序訊號跨期穩定性的指標，GAT 中性化系列在長期視窗的表現已明顯優於基本 GAT 變體，亦優於不使用圖結構的 DNN。

為驗證上述 IC 優勢的統計顯著性，針對長期視窗 GAT-IndNeutral 的逐日 IC 序列進行正式統計檢定。

\textbf{Newey-West HAC $t$ 檢定。}截面 IC 序列並非獨立同分布（i.i.d.）。由於同一批股票在相鄰交易日之間共享部分資訊環境，IC 序列存在顯著的序列自相關（長期視窗一階自相關約 0.53），若以普通最小平方法的 naive 標準誤計算 $t$ 統計量，將因忽略自相關而系統性高估顯著性。Newey-West 異質性-自相關一致（HAC）估計量~\cite{newey1987simple}在計算標準誤時納入 $L$ 個滯後期的自協方差加權修正，本文依 Andrews（1991）~\cite{andrews1991autocovariance}自動選階規則設定 lag $= 5$，相當於覆蓋約一週的交易日序列依存結構。修正後，GAT-IndNeutral 的 NW $t = 11.76$（$p < 0.001$），DNN 為 $t = 10.83$（$p < 0.001$）；兩者均在高度顯著水準下拒絕「平均 IC 為零」的虛無假說，確認長期視窗的截面排序訊號具有穩健的統計意義，而非僅因樣本期內少數極端月份所致。

\textbf{Block Bootstrap ICIR 信賴區間。}ICIR 為 IC 均值除以 IC 標準差，其抽樣分布在有序列相關的資料中不服從標準常態，需以重抽樣法估計。採用環形 Block Bootstrap（block size $= 10$，重複 $B = 2000$ 次），在每次重抽樣中保留序列的局部時間依存結構，並計算 95\% 百分位信賴區間。結果：GAT-IndNeutral 之 ICIR 95\% CI 為 $[0.624,\ 0.957]$，下界遠大於零，表明排序訊號的跨期一致性（而非單純均值）本身即達顯著水準；DNN 之 CI 為 $[0.563,\ 0.901]$，同樣顯著但區間下界略低、寬度略寬，對應於其逐日 IC 方差相對較大的特性。

\textbf{月度勝率。}從月度維度觀察，長期視窗測試期共 28 個月中，GAT-IndNeutral 有 27 個月 IC 為正（月度勝率 96.4\%，月度 ICIR $= 1.866$）；DNN 則達到 28/28 月（100\%），但月度 ICIR 為 1.763，低於 GAT-IndNeutral。此結果說明 GAT-IndNeutral 雖偶有單月小幅負 IC，但其平均逐日 IC 的跨月穩定性更高，兩者互為補充：DNN 的勝率更穩但每日訊號波動較大，GAT-IndNeutral 的平均訊號強度更高。

\subsection{因子注意力特徵之描述性分析}
\label{subsec:attention_analysis}

\section{綜合詮釋與研究限制}

% =========================================================

\subsection{評估指標選擇之方法論考量}
\label{subsec:metric_discussion}

本實驗顯示，均方根誤差（RMSE）不適合作為跨架構的唯一比較指標。以長期視窗為例，線性迴歸、LSTM 與 GAT-industry 的 RMSE 約為 0.0484 至 0.0485，而 DNN、GAT-IndNeutral 與 GAT-full 的 RMSE 分別為 0.1336、0.1160 與 0.1817；然而後三者在 Daily IC、ICIR 與 Sharpe 比率上均明顯優於前者。RMSE 偏高的原因在於因子注意力與中性化模型的輸出尺度與原始報酬空間不完全一致，且截面排序任務所需者為相對強弱排序而非精確數值預測。此結果印證以 IC/ICIR 為主要評估指標的方法論選擇：排序訊號的跨期穩健性比均方誤差更直接反映模型在橫斷面投資策略中的可用性。

\subsection{圖模型增量貢獻之定位}
\label{subsec:graph_contribution}

綜合本章各節的分析，圖注意力機制的增量貢獻應定位於訊號穩定性、拓樸可解釋性與截面辨識力的提升，而非對非圖模型的全面性績效主導。DNN 在短期與中期視窗已展現具競爭力的截面排序能力，GAT 系列與 GAT 中性化系列的優勢主要體現於中、長期視窗的 ICIR 穩定性，以及使用具有經濟意義之產業圖拓樸時的截面差異保留。GAT-universe 的結果則說明，圖的引入若不搭配合理的拓樸設計，反而因過度平滑化降低個股截面辨識力。

為量化上述增量貢獻的統計顯著性，本文針對長期視窗進行 Diebold-Mariano（DM）配對比較檢定，以直接回答「圖模型的截面排序能力是否統計上優於非圖基準」的核心問題。

\textbf{Diebold-Mariano 檢定。}DM 檢定~\cite{diebold1995dm}以逐期損失差序列 $d_t = L(\hat{y}^A_t, y_t) - L(\hat{y}^B_t, y_t)$ 為基礎，虛無假說為 $H_0: \mathbb{E}[d_t] = 0$（兩模型預測能力相等）。本文採用截面排名相關係數（IC）的負值作為損失函數，即 $L = -\text{IC}_t$，$d_t = \text{IC}^B_t - \text{IC}^A_t$ 表示模型 A 每日相對 B 的 IC 超額。由於 $d_t$ 序列同樣存在序列自相關，DM 統計量的標準誤以 Newey-West HAC 估計量計算，確保推論的有效性。

結果以 DNN 為非圖基準，對長期視窗各 GAT 模型進行配對檢定，呈現明確的分層（各模型數值可對照表~\ref{tab:results_overview_master}）：
\begin{itemize}
  \item \textbf{GAT-IndNeutral 對 DNN}：平均每日 IC 差 $+0.0055$，DM 統計量 $= 2.330$（$p = 0.010$），在 5\% 顯著水準下拒絕 $H_0$。
  \item \textbf{GAT-full 對 DNN}：DM 統計量顯著（$p = 0.031$），確認中性化延伸模型均具有統計可辨識的排序能力增益。
  \item \textbf{純圖模型（GAT-industry、GAT-universe、GAT-TwoGraph）對 DNN}：DM 檢定均未達顯著水準（$p > 0.05$），儘管其 ICIR 在點估計上高於或接近 DNN。
\end{itemize}

這一分層結果具有重要的方法論意涵：圖拓樸結構的引入，若不搭配中性化機制，其所帶來的絕對 IC 提升在統計上無法與 DNN 區分。換言之，\textbf{中性化機制——而非圖連結本身——才是可統計驗證的增量來源}。此發現將「圖模型的貢獻」定位於訊號的精煉與去噪（中性化約束），而非單純的鄰域資訊聚合。

\textbf{訓練集與測試集指標對照。}表~\ref{tab:results_overview_master}「比值」欄位揭示各模型泛化能力的系統性差異。短期視窗中，Linear、XGBoost 與 LSTM 的訓練/測試 IC 比值均僅為 0.06，為極端過擬合；XGBoost 訓練 ICIR 高達 5.71，測試 ICIR 僅 0.21，差距為全模型之最。隨視窗延長，比值整體提升：長期視窗中 GAT-IndNeutral（0.38）與 DNN（0.38）並列最高，在達到更高絕對 IC 的同時維持相近的泛化效率；XGBoost 長期比值（0.16）仍顯著偏低，其高報酬數字應合併診斷解讀。

\subsection{研究限制}
\label{subsec:limitations}

本實驗存在若干重要限制，應於解讀結果時一併考慮。

其一，回測尚未納入交易成本、市場衝擊與換手率約束，現有績效數字應視為無摩擦假設下的上限估計，實際可落地的超額報酬將低於本文所報告的數值。

其二，本文已針對長期視窗 GAT-IndNeutral 與 DNN 進行 Newey-West $t$ 檢定（$t = 11.76$，$p < 0.001$）與 Diebold-Mariano 配對比較（$p = 0.010$），以及 Block Bootstrap ICIR 信賴區間估算（95\% CI = [0.624, 0.957]）。然而，短期與中期視窗及其餘模型組合的逐一統計檢定、分年度情境分析與跨市場條件分組驗證，有待後續研究進一步補充，以更全面評估各模型在不同市場環境下的跨期一致性。

其三，本文圖結構所使用之產業分類為穩定外生資訊，不涉及前視偏誤；但若未來研究採用動態圖（如滾動相關係數）構建邊，則需嚴格確保圖建構僅使用截至當前交易日的訓練期資訊，以避免前視偏誤影響評估結果。
