本研究探討在台灣股票市場之橫斷面報酬預測中，模型結構、圖拓樸與訓練視窗長度如何影響 rank signal 與投資組合表現。相較於單純檢驗「GAT 是否優於 baseline」，本文將圖注意力模型定位為關係結構與因子表示的消融實驗，並同時檢視預測層級的 IC / ICIR 與投資組合層級的風險調整報酬。

實證資料採用 TEJ 台股日頻資料，建構 56 維技術與估值特徵，標籤為未來 5 日報酬之每日截面去均值。實驗建立產業圖與全市場完全圖，並比較 Linear、XGBoost、LSTM、DNN、GAT-industry、GAT-universe、GAT-TwoGraph、GAT-IndNeutral 與 GAT-full。本文設計 short、medium 與 long 三種訓練視窗，以時間序列方式切分訓練、驗證與測試集；策略層面則每 5 個交易日依預測分數排序，選取前 10\% 股票等權做多，並與 0050 基準比較。

研究結果顯示，訓練視窗長度對 out-of-sample 可用性具有主導性影響。Short window 雖可產生部分正向 Daily IC，但多數投資組合年化報酬接近 0 或為負；medium 與 long window 方能穩定轉化為正向 Sharpe。Long window 下，GAT neutralization family 的平均 Daily IC 為 0.0587、平均 Sharpe 為 1.826，高於 GAT family 的 0.0502 與 1.704，亦高於 DNN 的 0.0535 與 1.564。圖拓樸方面，GAT-industry 在 medium 與 long window 的 Daily IC 與 Sharpe 均優於 GAT-universe，顯示具有明確經濟意義的產業圖比全市場完全連結更能保留個股截面差異。GAT-IndNeutral 在 long window 取得最高 Daily IC（0.0590）與 ICIR（0.786），GAT-full 則取得最高 Sharpe（1.849）與 42.02\% 年化報酬，兩者呈現純 alpha 訊號穩定性與絕對報酬之間的 trade-off。XGBoost 在 long window 總報酬達 143.08\%，但 Train IC 與 Test IC 分別為 0.7824 與 0.0205，顯示其存在顯著過擬合風險。整體而言，圖模型的貢獻應被定位為訊號穩定性、拓樸選擇與可解釋性的提升，而非對非圖模型的全面性主導優勢。
