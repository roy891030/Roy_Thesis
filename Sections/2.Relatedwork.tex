\chapter{文獻回顧}
\label{ch:relatedwork}

本章依本文之研究問題整理文獻脈絡。本文關注的不是單一模型是否能降低預測誤差，而是在統一資料切分與評估框架下，檢驗技術與多因子特徵、時間序列資訊、股票關係圖、message passing 口徑與中性化設計，分別對截面報酬排序與投資組合績效的增量貢獻。因此，本章依序回顧截面報酬預測、特徵來源、非圖與序列基準、股票關係圖、GNN / GAT 訊息傳遞、多因子中性化與消融實驗，以及 IC / ICIR 與投資組合驗證。

\section{截面報酬預測與股票選取問題}
\label{sec:cross_sectional_return_stock_selection}

股票預測任務可分為價格水準預測、方向預測與截面報酬排序。價格水準預測關注單一資產未來價格，較容易受到不同股票價格尺度、除權息與市場共同漲跌影響；截面報酬預測則在同一交易日比較多檔股票的相對強弱，目標是形成可用於股票選取的排序訊號。Fama 與 French \cite{fama1992cross_section,FAMA19933} 以規模、帳面市值比與共同風險因子說明平均報酬的截面差異；Jegadeesh 與 Titman \cite{jegadeesh1993returns} 則指出動能訊號可在截面上區分過去贏家與輸家。這些文獻奠定了以股票特徵解釋相對報酬差異的基礎。

近年機器學習資產定價研究進一步強調，高維 firm characteristics 與非線性模型可用於預測 expected returns。Gu、Kelly 與 Xiu \cite{gu2020empirical} 顯示樹模型與神經網路能捕捉傳統線性模型較難表示的交互作用；Freyberger、Neuhierl 與 Weber \cite{freyberger2020dissecting} 亦指出部分特徵對截面預期報酬的影響具有非線性與條件性；Kozak、Nagel 與 Santosh \cite{kozak2020shrinking} 則從高維特徵與因子模型角度討論如何壓縮截面資訊。這些研究支持本文將模型定位為截面排序器，而不是單純的價格點預測器。

本文採用未來五日報酬之每日截面去均值作為訓練標籤。使用 future return 而非 future price，是因為投資決策關注可實現報酬而非價格水準；使用每日截面去均值，是為了降低市場共同漲跌對標籤的支配，使模型更聚焦於同一日股票之間的相對表現。五日 horizon 可視為短週期報酬預測與週頻再平衡之間的折衷，既避免單日報酬過度受微結構噪音影響，也不將持有期拉長到使訊號與圖結構反應過慢。Abe 與 Nakagawa \cite{abe2020cross_sectional_daily} 在實際投資管理情境中亦使用 next five-day stock return 作為 cross-sectional prediction 的目標，說明此類短週期 forward return 可直接銜接投資組合建構。

\section{技術指標、多因子特徵與非圖 baseline}
\label{sec:technical_multifactor_non_graph_baselines}

技術指標將價格、成交量與交易區間資訊轉換為可計算特徵，長期被用於交易規則與報酬預測。Brock、Lakonishok 與 LeBaron \cite{Brock1992SimpleTT} 顯示移動平均與區間突破規則在長期樣本中具有統計可辨識性；Lin \cite{LIN2018103} 則指出，技術指標經適當對齊與降噪後可改善樣本內與樣本外預測。Li 與 Bastos \cite{li2020stock}、Sezer 等人 \cite{sezer2019dl_fin_survey} 與 Sonkavde 等人 \cite{sonkavde2023review_ml_dl} 的整理亦顯示，技術指標常作為深度學習金融預測的結構化輸入。

需要區分的是，technical indicators、firm characteristics 與 learned factors 在文獻脈絡上並不完全相同。Technical indicators 通常指由價量資料衍生的趨勢、動能、波動或量能訊號，例如 RSI、MACD、KD 或 Bollinger Bands；firm characteristics 則指公司層級或資產定價文獻中的特徵，例如 size、book-to-market、profitability、leverage 或 liquidity；learned factors 則是模型由輸入特徵學出的 latent representation。本文所稱多因子特徵，並非完全等同於傳統資產定價中的已命名風險因子，而是包含技術指標、價量特徵與部分財務／估值變數的廣義 firm-level predictors。此定義與機器學習資產定價文獻將大量 characteristics 作為 expected return predictors 的做法一致 \cite{gu2020empirical,freyberger2020dissecting,kozak2020shrinking}。

非圖 baseline 的用途，是在不使用股票關係圖的條件下建立比較基準。Ridge regression 可視為低容量、可解釋且具正則化的線性基準；Hoerl 與 Kennard \cite{hoerl1970ridge} 指出，Ridge 透過 \(L_2\) 懲罰能緩解共線性造成的係數不穩定。由於技術與多因子特徵常具有高度相關與重複訊號，Ridge 適合作為檢查特徵基本線性預測力的下限。

樹狀集成模型則提供非線性 tabular benchmark。Chen 與 Guestrin \cite{Chen2016XGBoost} 提出的 XGBoost 透過梯度提升、正則化、欄位抽樣與稀疏特徵處理提升結構化資料建模能力。金融預測相關研究亦常使用 Decision Tree、Random Forest、AdaBoost 或 XGBoost 作為非線性比較模型 \cite{ampomah2020tree_direction,ampomah2020tree_ensemble_stock,nti2020ensemble_comprehensive,saifan2020ensemble_trading,WU2006270}。因此，若圖模型無法穩定超越強 tabular baseline，其圖結構增益就不能只用「深度模型較複雜」解釋，而必須回到關係圖、message passing 與中性化設計本身檢驗。

\section{時間序列模型與單股歷史訊號限制}
\label{sec:time_series_single_stock_limit}

時間序列模型用於檢查單股歷史路徑是否足以提供增量訊號。長短期記憶網路（Long Short-Term Memory, LSTM）透過 forget、input 與 output gates 控制長短期資訊流，能緩解傳統 RNN 的長序列梯度消失問題 \cite{hochreiter1997lstm}。在股票預測文獻中，Qiu 等人 \cite{qiu2020forecasting} 與 Zaheer 等人 \cite{zaheer2023multi} 將 LSTM 或其變形應用於股票價格與多維價格訊號預測，顯示序列模型能吸收單一資產的時間依賴。

\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.92\textwidth]{img/LSTM_graph.jpg}
  \caption{LSTM 單元的資訊流（Forget / Update / Output）。其中 $x_t$ 為當期輸入，$h_{t-1}$ 為前一期隱狀態（hidden state），$c_{t-1}$ 為前一期記憶狀態（cell state）。}
  \label{fig:lstm_cell_flow}
\end{figure}

LSTM 的優勢在於能以 sequence-to-one 的方式吸收單股過去價格、成交量與技術特徵路徑。然而，若研究任務是每日股票排序，單股序列模型的限制也很明顯：它主要處理同一股票內部的時間依賴，不直接刻畫同一交易日其他股票對目標股票表示的影響。當市場存在產業共振、全市場風格切換、流動性輪動或短期共動結構時，單股序列模型只能透過人工特徵間接吸收這些訊號。這也是後續股票關係圖與 GNN 方法被引入截面股票預測的主要理由。

\section{股票關係圖：產業圖、市場圖、相關係數圖}
\label{sec:stock_relation_graph_industry_market_corr}

金融市場中的股票並非彼此獨立。個股報酬可能共同受到產業景氣、風格因子、總體市場風險偏好、資金流向與價格共動結構影響。股票關係圖提供一種將股票視為節點、將股票間關係視為邊的資料結構，使模型能在同一交易日聚合其他股票資訊。既有研究常以產業關係、供應鏈、價格相似性、知識圖譜或動態關係建立股票圖，並觀察其對報酬預測或股票排序的影響 \cite{song2023stock,xiang2022dynamic_gat,su2024hetero_gat,huang2022mlgat,wei2025fstgat}。

產業圖（industry graph）將同一產業內股票連結，用以表示產業共同成分與同業間資訊傳遞。其優點是經濟意義清楚、結構穩定且可重現；限制是無法捕捉跨產業共動與同產業內關係強度變化。市場圖（market graph）若採全截面連結，則不應被解讀為用來預測整體市場方向。尤其在本文標籤已做每日截面去均值後，市場共同方向已被部分移除；因此，市場圖更合理的定位，是一種全截面資訊聚合機制，用以檢驗模型是否能透過全市場節點互動捕捉當日共同風格、流動性或資金輪動訊號。其限制是全連接結構接近 global pooling，可能混入大量弱關係與噪音邊。

相關係數圖（correlation graph）以 rolling return correlation 建立時間變動邊，使邊的存在與權重可隨市場共動變化而更新。Mantegna \cite{mantegna1999hierarchical} 與 Onnela 等人 \cite{onnela2003dynamics} 說明相關係數網路可用於描述金融市場共動結構與其時間變化；Engle、Ledoit 與 Wolf \cite{engle2019large_dynamic_covariance} 則從大型動態共變異矩陣角度顯示，金融資產間的相關與共變異估計本身具有高維與時間變動挑戰。

因此，dynamic correlation graph 的價值與限制必須同時說明。Rolling window 太短時，相關係數估計較不穩定；rolling window 太長時，圖結構反應市場狀態變化較慢。Top-\(k\) 稀疏化會改變 graph topology，使模型對鄰居選取規則敏感；使用 signed correlation 或 absolute correlation 也會導致不同經濟解讀。這些限制說明，dynamic graph 不應被預設為必然優於 static graph，而應在相同資料切分、模型容量與評估口徑下實證檢驗。

\section{GNN / GAT message passing 在股票網路中的角色}
\label{sec:gnn_gat_message_passing_stock_network}

圖神經網路（Graph Neural Networks, GNNs）以圖作為基本資料結構，透過節點與邊描述實體及其關係，並以 message passing 聚合鄰居資訊。Scarselli 等人 \cite{scarselli2009gnn} 提出早期 GNN 架構；Wu 等人 \cite{wu2020comprehensive} 則系統整理後續圖卷積與訊息傳遞模型。PyTorch Geometric 所提供的圖資料表示與圖層實作，亦使大規模股票圖建模具備可操作性 \cite{fey2019pyg}。

\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.96\textwidth]{img/architecture_of_GNN.png}
  \caption{GNN 架構示意：由輸入層 $X$ 經過多層圖卷積/訊息傳遞後得到輸出層表示 $Z$。}
  \label{fig:gnn_architecture}
\end{figure}

在股票網路中，message passing 可解讀為每檔股票根據自身特徵形成節點表示後，再從其鄰居股票吸收關係訊號。若圖為產業圖，鄰居聚合主要反映同業共同成分；若圖為市場圖，鄰居聚合偏向全截面共同資訊；若圖為相關係數圖，鄰居集合與關係強度會隨近期價格共動更新。此流程使模型能將「個股特徵」與「股票間關係」同時納入表示學習。

圖注意力網路（Graph Attention Network, GAT）在鄰居聚合過程中加入 attention mechanism，使目標節點可對不同鄰居分配不同權重 \cite{velickovic2018gat}。相較於固定平均聚合，GAT 更適合處理金融圖中的異質關係，因為股票之間的影響強度可能隨產業、流動性、市場狀態與共同波動而不同。

\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.95\textwidth]{img/gat_attention_mechanism.png}
  \caption{GAT 的鄰居注意力機制示意。左圖呈現注意力係數 $\alpha_{ij}$ 的計算流程；右圖示意節點 $i$ 對其鄰居集合 $\mathcal{N}(i)$ 分配不同權重後進行聚合。}
  \label{fig:gat_attention_mechanism}
\end{figure}

需要注意的是，GAT attention 的解釋對象主要是 node-to-node aggregation weight，也就是哪些鄰居股票在表示更新時被賦予較高權重。若模型另設 feature attention 或 factor attention，其解釋對象則是特徵或 latent factor 層級，兩者不應混用。Jain 與 Wallace \cite{jain2019attention_not_explanation} 以及 Serrano 與 Smith \cite{serrano2019attention_interpretable} 均提醒，attention weights 可作為模型行為的描述性線索，但不必然等同嚴格解釋或因果效果。因此，本文後續若分析 attention，應定位為模型診斷與輔助觀察，而非單獨支持經濟因果結論。

\section{多因子表示、中性化與消融實驗邏輯}
\label{sec:multifactor_neutralization_ablation_logic}

多因子模型的核心目標，是將大量原始特徵轉換為較高層次的投資訊號表示。傳統因子模型常以線性暴露解釋股票報酬差異；深度多因子模型則進一步讓模型從高維特徵中學習非線性因子表示。Wei、Dai 與 Lin \cite{wei2022factor} 提出的 Deep Multi-Factor Model 以深度表示、因子收益與注意力解釋模組結合，提供本文設計多因子表示架構的重要參考。

在量化股票選取中，中性化的目的通常是降低策略對非目標風險來源的暴露，例如產業、規模、價值或市場 beta。Ehsani、Harvey 與 Li \cite{ehsani2023sector_neutrality} 討論 sector neutralization 在 factor investing 中的效果，指出中性化可能有助於分離 stock selection 與 sector bet，但其效果也會依投資組合型態而異。這提醒本文不能只宣稱「中性化必然更好」，而應把中性化視為一個需要實證檢驗的模型設計。

同時，本文之中性化屬於 representation-level neutralization，概念上借鑑傳統因子中性化，但並非直接對報酬或原始特徵進行橫截面迴歸殘差化。傳統 residualization 通常透過迴歸移除特定暴露；表示層中性化則在神經網路 latent representation 中嘗試分離個股語境、產業共同成分與市場共同成分。因此，第 2 章只需說明中性化與消融實驗的必要性，具體模型命名、資料流與數學形式應在方法章定義。

消融實驗是辨識增益來源的必要設計。若只比較一個完整圖模型與一個 baseline，無法判斷提升究竟來自非線性特徵編碼、時間序列記憶、產業圖、全市場圖、動態相關係數圖、注意力機制或中性化設計。更嚴謹的作法，是在相同資料、相同切分、相同訓練流程與相近模型容量下，逐步加入或移除關鍵模組。如此才能判斷模型改善是否真正來自特定設計，而不是來自資料範圍、訓練設定或模型複雜度差異。

\section{IC / ICIR 與投資組合績效驗證}
\label{sec:ic_icir_portfolio_validation}

由於本文任務是股票排序而非單點價格預測，因此 ranking quality 比 absolute error 更貼近投資決策。RMSE / MAE 衡量預測值與真實值的數值距離，但 IC / ICIR 衡量模型分數是否能在每日截面上正確排序股票。資訊係數（Information Coefficient, IC）通常定義為模型分數與未來報酬在同一交易日股票截面上的相關係數；Zhang、Guo 與 Cao \cite{zhang2020information_coefficient} 即將 IC 視為 stock selection model 的績效衡量方式。Ding 與 Sun \cite{ding2022timevarying_ic} 討論 time-varying cross-sectional IC 的統計性質；Qian 與 Hua \cite{qian2004active_risk} 則將 IC 與投資組合資訊比率連結，指出訊號品質與有效下注寬度共同決定策略績效。

然而，截面選股模型即使在 IC 上表現良好，也不必然能產生可投資績效。原因在於投資組合績效同時受到排序前段股票的集中度、再平衡頻率、波動、回撤、交易成本與換手率影響。Sharpe \cite{sharpe1994ratio} 提出的 Sharpe ratio 是常用的風險調整後報酬指標；但在策略比較中，若只報告高 Sharpe 而未考慮樣本長度與多重測試，容易高估策略穩健性。

投資實證文獻亦指出，交易成本與換手會顯著影響訊號可實現性。Ye \cite{ye2008signal_quality} 與 Zhang 等人 \cite{zhang2021turnover_ir} 討論訊號品質、換手與資訊比率之間的關係；Keim 與 Madhavan \cite{keim1997transaction_costs} 以及 Novy-Marx 與 Velikov \cite{novy_marx2016taxonomy} 則說明不同交易風格與異常報酬在扣除交易成本後可能大幅衰減。因此，若策略採用高頻再平衡或選入流動性較差股票，無摩擦回測結果應被視為上限估計。

此外，多模型、多期間、多指標比較容易產生 data snooping 與 selection bias。White \cite{white2000reality_check} 提出 Reality Check 以處理資料探勘下的模型比較；Bailey 與 Lopez de Prado \cite{bailey2014deflated_sharpe} 討論 Deflated Sharpe Ratio，用於修正策略選擇偏誤與非正態報酬；Harvey、Liu 與 Zhu \cite{harvey2016cross_section} 則指出大量因子檢定會提高偽陽性風險。這些文獻提醒，本文後續呈現多組統一實驗結果時，應將 winner 解讀為同一實驗設定下的條件性結果，而非未經檢定的普遍最優策略。

在 rolling portfolio validation 上，較嚴謹的作法是於每個再平衡日使用當期模型分數排序股票，選取固定比例或固定檔數股票，等權或依權重持有至下一再平衡日，並將策略報酬與 benchmark 在完全相同日期、相同持有期下對齊。此流程可將預測層訊號轉換為投資層績效，但仍須明確說明交易時點、持有期、再平衡頻率、交易成本與 benchmark 對齊方式。

\section{本研究缺口與總體實驗定位}
\label{sec:research_gap_overall_experiment_positioning}

綜合上述文獻，既有研究已分別討論技術指標、firm characteristics、非線性機器學習、LSTM 序列模型、GNN / GAT 股票圖、多因子表示與投資組合績效。然而，與本文總體實驗直接相關的缺口仍然存在。

第一，許多研究將模型改善歸因於深度學習或圖注意力，但未必同時納入強 tabular baseline、序列 baseline 與圖模型消融。因此，若未控制資料切分、baseline 與模型容量，便難以判斷模型改善是否真正來自圖關係建模，而非模型複雜度或訓練設定差異。第二，部分圖模型研究使用單一 static graph，較少在相同資料、相同切分與相同架構下比較產業圖、市場圖與 rolling correlation dynamic graph；若未控制這些條件，便難以判斷 dynamic graph 的改善是否真正來自動態關係建模，而非資料或模型差異。第三，注意力權重常被用於解釋模型，但若未區分 node-to-node attention 與 feature / factor attention，容易造成解釋對象混淆。第四，許多預測研究偏重 error metrics，未同步檢查 IC、ICIR 與 rolling portfolio validation 是否一致。

因此，本文的不可替代性在於：於同一 TEJ 台股日頻資料、相同時間切分、相同評估流程與同一投資組合驗證邏輯下，系統性比較技術與多因子特徵、時間序列資訊、股票關係圖、static / dynamic message passing 與中性化設計對截面排序與投資組合績效的增量貢獻。此設計的目的不是證明單一模型永遠最佳，而是辨識不同建模元件在不同期間與不同評估層級下的條件性價值。
