\normalsize

This study examines whether graph attention networks improve stock selection and return prediction in the Taiwan stock market. Using daily TEJ data, we build 54 technical features, an industry graph, and a market-wide graph, and compare five models: Linear Regression, XGBoost, LSTM, GAT, and DMFM within a reproducible pipeline for data processing, training, and evaluation. Performance is evaluated by RMSE, MAE, IC, ICIR, and directional accuracy, while portfolio validation ranks stocks at each rebalancing date, takes equal-weight long positions in the top names, and compares the results with the 0050 benchmark.

Results show that DMFM provides the strongest overall signal quality and portfolio performance. In the Long window, ICIR improves from 0.3865 under Linear Regression to 0.7854 under DMFM. Relative to 0050, annualized return increases from -3.35\% to 31.31\%, Sharpe ratio rises from -0.113 to 1.369, and win rate improves from 48.6\% to 62.9\%. Overall, graph attention modeling improves the usability and interpretability of stock-selection signals for practical quantitative investing.
