    \textbf{Abstract} \\[0.5cm]

\end{center}

\normalsize

This study examines whether graph attention and multi-factor representations improve stock selection and return prediction in the Taiwan stock market. Using daily TEJ data, we build 56 technical features and define the training target as the cross-sectionally demeaned five-day forward return. The unified experiment pipeline compares Linear Regression, XGBoost, and LSTM with six factor ablation models: MLP, GAT-Industry, GAT-Universe, GAT-TwoGraph, DMFM-IndNeutral, and DMFM-Full. The graph experiments are evaluated under both static graphs and dynamic weighted graphs. Static graphs use same-industry complete connections and a market-wide complete graph, while dynamic graphs use rolling return correlations with edge attributes for absolute and signed correlation. Prediction performance is evaluated by MSE, RMSE, MAE, IC, Daily IC, ICIR, and directional accuracy. Portfolio validation ranks stocks every five trading days, takes equal-weight long positions in the top 10\%, and compares the results with the 0050 benchmark.

The results show substantial heterogeneity across sample windows, graph modes, and evaluation layers. In the Short/static setting, GAT-Universe achieves the highest Test IC (0.0843), total return (32.13\%), and Sharpe ratio (7.832). In the Medium/dynamic setting, GAT-Universe obtains the highest total return (38.31\%) and Sharpe ratio (1.866). In the Long/static setting, DMFM-Full achieves the highest total return (54.32\%) and Sharpe ratio (1.490). Across 18 paired dynamic-versus-static comparisons, dynamic graphs improve Sharpe ratios in 10 cases and total returns in 11 cases, but reduce Test IC by 0.0022 on average, indicating that dynamic graph construction is conditional rather than uniformly superior. Averaged across available experiments, GAT-Industry has the highest mean Sharpe ratio (3.438), DMFM-Full has the highest mean total return (36.40\%), and GAT-Universe has the highest mean Test IC (0.0858). Attention analysis further shows that rolling min/max, volatility, and volume-related features appear most frequently among the top-ranked signals. Overall, graph attention and multi-factor representation can improve cross-sectional ranking and portfolio performance, but their value must be assessed jointly through model ablations, graph construction choices, and portfolio-level metrics.
