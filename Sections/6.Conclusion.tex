\chapter{結論與未來工作}
\label{ch:conclusion}

本文以 TEJ 台股日頻資料建立 56 維技術與估值特徵，在統一實驗框架下比較線性迴歸、XGBoost、LSTM、DNN、圖注意力拓樸系列與圖注意力中性化系列共九類模型。實證結果顯示，台股橫截面報酬預測的關鍵在於訓練視窗長度、圖拓樸設計與模型結構如何共同影響截面排序訊號品質與投資組合績效，而非單純比較特定模型架構是否具備整體優越性。

第一個主要發現是，長期訓練視窗顯著提升模型在測試集的可用性。短期視窗雖能產生部分正向 Daily IC，但多數模型的投資組合年化報酬接近零或為負；中、長期視窗才穩定轉化為正向 Sharpe 比率。長期視窗下，GAT 中性化系列的平均 Daily IC 為 0.0587、平均 Sharpe 比率為 1.826；GAT 拓樸系列為 0.0502 與 1.704；DNN 為 0.0535 與 1.564。較長的訓練樣本能覆蓋更多元的市場狀態，使模型習得的截面排序訊號更具跨期穩健性。

第二個主要發現是，圖拓樸設計的選擇對模型表現具有決定性影響，其重要性高於圖結構是否存在本身。GAT-industry 在中、長期視窗的 Daily IC 與 Sharpe 比率均優於 GAT-universe，支持以具明確產業關聯意義的圖拓樸為基礎建構模型。相對地，全市場完全連結圖容易引入市場共同成分，其長期視窗 Daily IC 與 Sharpe 比率甚至低於不使用圖結構的 DNN。此結果顯示，圖模型的效益取決於拓樸能否有效保留個股截面差異，而非訊息傳遞機制本身。

第三個主要發現是，圖注意力中性化系列在訊號品質指標上表現最為穩定。中期視窗中，GAT-full 取得全模型最高 Daily IC（0.0596）與 ICIR（0.628）；長期視窗中，GAT-IndNeutral 取得最高 Daily IC（0.0590）與 ICIR（0.786），GAT-full 則取得最高 Sharpe 比率（1.849）與年化報酬（42.02\%）。兩者之間存在可觀察的取捨關係：GAT-IndNeutral 以較高的 ICIR 反映純截面選股訊號的穩定性，而 GAT-full 因保留較高程度的產業與市場曝險，在有利市場環境下可轉化為較高的投資組合絕對報酬。

此外，基準模型的結果值得正視。DNN 在短、中期視窗取得相當具競爭力的 IC 與 Sharpe 比率，顯示 56 維技術與估值特徵本身已提供可觀的截面排序訊號；圖模型的貢獻應定位為長視窗下的訊號穩健性、拓樸可解釋性與結構性改善，而非對非圖模型的全面優勢。XGBoost 在長期視窗的總報酬達 143.08\%，但 Daily IC 測試/訓練比僅為 0.16，顯示其投資組合報酬與樣本外排序訊號品質並不一致，不宜以單一報酬指標概括其預測品質。

未來研究將聚焦於下列工作：

\begin{itemize}
  \item \textbf{納入交易成本、滑價與換手控制：}本文目前之回測仍屬無摩擦市場設定；後續可將手續費、滑價與換手率懲罰納入，並進一步評估換手調整後的 Sharpe 比率與資訊比率，以確認超額報酬在實務落地後的可持續性。
  \item \textbf{擴充統計穩健性檢定範圍：}本文已完成長期視窗之 Newey-West HAC $t$ 檢定、Block Bootstrap ICIR 信賴區間與 Diebold-Mariano 配對比較；後續可將相同檢定擴充至短期與中期視窗及其餘模型組合，並加入滾動視窗穩健性檢定。
  \item \textbf{擴充關係圖對照：}後續可加入滾動相關係數圖、成交量共動圖、基本面相似度圖或供應鏈關係圖等其他建構方式，並檢驗其是否在不引入估計噪音的情況下改善 ICIR 與投組風險。
  \item \textbf{補充分年度與市場狀態分析：}可依牛市、熊市、震盪期或利率環境分組檢查 IC 與投組績效，特別觀察 GAT-full 與 GAT-IndNeutral 在不同市場狀態下的產業曝險取捨。
\end{itemize}
