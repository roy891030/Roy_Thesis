\chapter{結論與未來工作}
\label{ch:conclusion}

本文以 TEJ 台股日頻資料建立 56 維技術與估值特徵，在統一框架下比較線性迴歸、XGBoost、LSTM、DNN、GAT 拓樸系列（GAT-industry、GAT-universe、GAT-TwoGraph）與 GAT 中性化系列（GAT-IndNeutral、GAT-full）共九類模型，涵蓋短期、中期、長期三個訓練視窗，評估指標以截面排序訊號品質（Daily IC、ICIR）為主，輔以投資組合績效（Sharpe 比率、年化報酬）。以下歸納四項主要發現。

\textbf{一、訓練視窗長度是可投資性的前提條件。}短期視窗（116 交易日）下，九類模型雖普遍產生正向 Daily IC，但投資組合測試總報酬均為負；中、長期視窗起，Sharpe 比率整體轉正，長期視窗最佳達 1.849。長期視窗的訓練期橫跨多種市場環境，使截面排序訊號具備更高的跨期穩健性。此結果提示，以短期回測評估圖模型效益，所得結論可能低估其長視窗下的實際表現。

\textbf{二、圖拓樸設計是必要非充分條件，具經濟意義的局部拓樸優於全市場完全圖。}GAT-universe 在長期視窗的測試 Daily IC（0.0452）與 Sharpe 比率（1.506）均低於不使用圖結構的 DNN（0.0535、1.564），顯示全截面完全連結易引入市場共同成分，導致過度平滑化而削弱截面辨識力。GAT-industry 以產業分類為邊，在長期視窗的 Daily IC（0.0543）與 Sharpe 比率（1.779）均優於 GAT-universe 與 DNN，支持以具明確經濟意義之局部拓樸為圖模型基礎。

\textbf{三、中性化機制是圖模型相對 DNN 唯一可統計驗證的增量來源。}透過 Newey-West HAC $t$ 檢定（lag $= 5$），長期視窗全部 GAT 模型與 DNN 的平均 IC 均在 $p < 0.001$ 水準下顯著大於零，Block Bootstrap 95\% 信賴區間下界亦均大於零，確認各模型排序訊號本身的統計可靠性。然而，Diebold-Mariano 配對比較（以 DNN 為基準）呈現明確分層：GAT-IndNeutral（DM $= 2.330$，$p = 0.010$）與 GAT-full（$p = 0.031$）對 DNN 的每日 IC 差異達 5\% 顯著水準；GAT-industry、GAT-universe 與 GAT-TwoGraph 則均未達顯著水準（$p > 0.05$），GAT-universe 的 DM 統計量甚至為負（$-1.983$）。此結果直接說明：\textbf{中性化機制——而非圖連結本身——是圖模型可統計驗證的增量貢獻}，純圖拓樸在訊號強度上無法與 DNN 區分。

\textbf{四、IC 與 Sharpe 衡量不同維度，非圖基準模型仍具競爭力。}GAT-IndNeutral 在長期視窗取得最高 ICIR（0.786），GAT-full 取得最高 Sharpe 比率（1.849），但中期視窗中 DNN 的 Sharpe 比率（0.980）與總報酬（39.13\%）仍高於 GAT 中性化系列，顯示 IC 排序領先並不自動轉化為 Sharpe 排序領先。Daily IC 衡量截面排序能力，Sharpe 涵蓋排序訊號轉化為持倉集中度與報酬的效率，需分層解讀。XGBoost 長期總報酬雖達 143.08\%，但 Daily IC 測試/訓練比值僅 0.16，顯示其報酬來自樣本內記憶效應而非可複製的截面排序訊號。

\noindent 未來研究建議聚焦以下方向：

\begin{itemize}
  \item \textbf{納入交易成本與換手控制：}現有回測為無摩擦市場假設，後續應納入手續費、滑價與換手率懲罰，評估實務落地後的換手調整 Sharpe 比率。
  \item \textbf{擴充統計檢定範圍：}本文已完成長期視窗的 NW $t$ 檢定、Bootstrap CI 與 DM 配對比較；後續可將相同檢定擴充至短期與中期視窗，並加入滾動視窗穩健性分析。
  \item \textbf{動態與多元圖結構：}後續可引入滾動相關係數圖、供應鏈關係圖或基本面相似度圖，並嚴格確保圖建構不引入前視偏誤，檢驗是否進一步改善 ICIR 與投組風險。
  \item \textbf{市場狀態分組分析：}依牛市、熊市、震盪期或利率環境分組，觀察 GAT-IndNeutral 與 GAT-full 在不同市場狀態下的 IC 穩定性與產業曝險取捨。
\end{itemize}
