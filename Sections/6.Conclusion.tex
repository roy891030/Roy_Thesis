\chapter{結論與未來工作}
\label{ch:conclusion}

本文以 TEJ 台股日頻資料建立 56 維技術與估值特徵，並在統一圖模型 pipeline 下比較 Linear、XGBoost、LSTM、DNN、GAT topology variants 與 GAT neutralization / factor-attention variants。實證結果顯示，台股橫斷面報酬預測的關鍵在於訓練視窗長度、圖拓樸與模型結構如何共同影響 rank signal 與投組表現，而非單純比較特定模型架構是否優越。

第一個主要發現是，長視窗訓練顯著提高 out-of-sample 可用性。Short window 雖能產生部分正向 Daily IC，但多數模型的投資組合年化報酬接近 0 或為負；medium 與 long window 才穩定轉化為正向 Sharpe。Long window 下，GAT neutralization family 的平均 Daily IC 為 0.0587、平均 Sharpe 為 1.826；GAT family 為 0.0502 與 1.704；DNN 為 0.0535 與 1.564。這表示較長樣本更能覆蓋市場 regime，使模型學到的橫斷面排序訊號更穩定。

第二個主要發現是，圖拓樸設計的選擇對模型表現具有決定性影響，其重要性高於圖結構是否存在本身。GAT-industry 在 medium 與 long window 的 Daily IC 與 Sharpe 均優於 GAT-universe，支持具有明確經濟意義之產業圖拓樸。相對地，全市場完全圖容易混入市場共同成分，其 long window Daily IC 與 Sharpe 甚至低於不使用圖結構的 DNN。此結果說明圖模型的效益取決於拓樸是否能有效保留個股截面差異，而非 message passing 機制本身。

第三個主要發現是，GAT 與 factor-attention 類模型在 signal quality 指標上表現最為穩定。Medium window 中 GAT-full 取得全模型最高 Daily IC（0.0596）與 ICIR（0.628）；Long window 中 GAT-IndNeutral 取得最高 Daily IC（0.0590）與 ICIR（0.786），GAT-full 則取得最高 Sharpe（1.849）與 42.02\% 年化報酬。此結果揭示兩者之間的 trade-off：GAT-IndNeutral 以較高的 ICIR 代表純 alpha 排序訊號的穩定性，而 GAT-full 因保留較多產業與市場 exposure，在有利市場環境下可轉化為較高的投組絕對報酬。

此外，baseline 模型的結果值得正視。DNN 在 short 與 medium window 取得相當具競爭力的 IC 與 Sharpe，顯示 56 維技術與估值特徵本身已提供可觀的截面排序訊號；圖模型的貢獻應被定位為長視窗下的穩定性、拓樸可解釋性與增量效益，而非對非圖模型的全面性主導優勢。XGBoost 在 long window 的總報酬達 143.08\%，但 Train IC 為 0.7824、Test IC 僅 0.0205，呈現顯著 train-test gap，說明其投組層級表現背後存在過擬合風險，不宜以單一報酬指標概括其預測品質。

未來研究將聚焦於下列工作：

\begin{itemize}
  \item \textbf{納入交易成本、滑價與換手控制：}本文目前之回測仍屬無摩擦市場設定；後續可將手續費、滑價與換手率懲罰納入，並進一步評估 turnover-adjusted Sharpe / IR，以確認超額報酬在實務落地後的可持續性。
  \item \textbf{補強統計穩健性檢定：}除現有 IC、ICIR 與投組績效比較外，後續可加入 paired t-test、bootstrap / block bootstrap、滾動視窗穩健性檢定與不同市場狀態分組分析，以更嚴謹地評估模型優勢是否具跨樣本穩定性。
  \item \textbf{檢查圖構建的 look-ahead 風險：}圖結構必須明確說明是否只使用訓練期資訊或穩定外生分類；若任何邊或權重使用全樣本資訊，需重新以訓練期資料建立圖並進行 robustness 檢查。
  \item \textbf{擴充關係圖對照：}後續可加入滾動相關、成交量共動、基本面相似度或供應鏈關係等其他圖建構方式，並檢驗其是否在不引入估計噪音的情況下改善 ICIR 與投組風險。
  \item \textbf{補充分年度與市場 regime 分析：}可依牛市、熊市、震盪期或利率環境分組檢查 IC 與投組績效，特別觀察 GAT-full 與 GAT-IndNeutral 在不同 regime 下的產業 exposure trade-off。
\end{itemize}
